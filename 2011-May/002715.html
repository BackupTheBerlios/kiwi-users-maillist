Hi,

> We are running kiwi v4.86.  We are building an oem iso/stick installer using the oemboot/suse-SLES11 boot config.  It doesn't look like the firmware files from kernel-default-base are being included in the initrd that is created.  When we boot we are getting error messages from the kernel as follows:
> bnx2: Can't load firmware file "bnx2/bnx2-mips-09-5.0.0.j9.fw"

You ran into that issue:

commit b70edb843bdde75f3f550dd6d457ca6fff93282a
Author: Marcus Schäfer <ms at novell.com>
Date:   Mon Apr 18 11:08:54 2011 +0200

    - fixed suseStripFirmware to match in subdirectories below
      /lib/firmware/ too. firmware installed by the kernel packages
      for example were removed

This fix is not part of the 4.86 release :( You might consider to
update your version. Alternatively I have attached the patch which
should fix this. Apply with

   patch -p0 < kiwi.firmware.patch

Sorry for any inconvenience

Regards,
Marcus
