That file must exist if kiwi deployed the image. So you have your
iso created by kiwi. What did you do to deploy the image ?
You can for example deploy the image on another VM like this:

   qemu-kvm -cdrom image.iso -boot d -hda mydisk

this would call qemu and install and boot onto mydisk. If I
login to that deployed machine I will have a log file in
/var/log/boot.kiwi.
