> >Hello, I was following the instructions from the KiWi Cookbook Get Juiced and ran into a error that I cannot seem to fix.  Here is the error from my terminal:
> >
> >linux-deqj:/home/mike/kiwi # mkdir /tmp/myjeos
> >linux-deqj:/home/mike/kiwi # kiwi --build suse-11.4-JeOS --destdir /tmp/myjeos --type iso
> >May-16 19:22:42<1>  : Reading image description [Prepare]...
> >May-16 19:22:42<2>  : Description provides no MD5 hash, check                                           skipped
> >May-16 19:22:42<3>  : Couldn't locate configuration directory suse-11.4-JeOS
> 
> This message is telling you that the directory suse-11.4-JeOS was not found.
> 
> Your kiwi command needs to point to the location of the
> configuration directory. If you did not make a copy of the template
> an modify it this should look as follows:
> 
> # kiwi --build /usr/share/kiwi/image/suse-11.4-JeOS -d /tmp/myjeos
> --type iso

This is not required. If you specify suse-11.4-JeOS only it will
automatically look if it can find in your current directory or in
/usr/share/kiwi/image.

I think you did not have installed the kiwi-templates package which
provides the templates. Can you check this ?

  rpm -ql kiwi-templates
