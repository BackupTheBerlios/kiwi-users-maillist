Hi,

> I have created a OEM image of OpenSuSE 11.4 with Split type.  Every thing is
> working fine except shutdown or halt.
> 
> I am not able to able to turnoff system / thinclient .
> 
> when I press shutdown button or halt command at then end off boot process i
> get error *"No more process left in this runlevel"* but at this stage
> thinclient should power off.

This might be related to the splitted root tree. If the root tree
can't be umounted because it's still busy for some reason the shutdown
process hangs here. You can write your own runlevel script which
kicks in before that happens and call "reboot -f" if it's safe

of course that's only a workaround. It would be good to know why
it hangs
