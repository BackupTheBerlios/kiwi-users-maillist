Marcus Schäfer wrote:
> Hi,
> 
>> When booting from a DVD created as an OEM image the menu title displays 
>> "Restore" as a prfix to the name. This is misleading as this is not a 
>> restore operation, it is an install. I think the prefix should be 
>> "Install" rather than "Restore".
> 
> We had that discussion in the past and the original name was "Install"
> but people use the same dvd to also restore the system to bring it
> back into a defined state. So if I change it back to "Install"
> the "restore" people will complain again :-)
> 
> Frankly said what should I do ? Change it to "Install/Restore" or
> something completely different ?

How about a new element:

<oem-name-prefix></oem-name-prefix>

The default could be "Restore". While it is very tempting to have this 
element be of type string for maximum flexibility, I'd say it should be 
of type token with "Restore" or "Install" as possible values.
