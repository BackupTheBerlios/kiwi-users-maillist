There are two possible ways:

1) make use of the new isohybrid technology (requires syslinux >= 3.82)
   http://lizards.opensuse.org/2009/08/05/hybrid-live-systems/

2) if you use an oem image make sure you have the following in
   your config.xml

   <oem-swap>no</oem-swap>
   <oem-systemsize>MB<oem-systemsize>
   <oem-home>no</oem-home>

   this will lead to a partition table which only uses the
   oem-systemsize bytes and leaves the rest free. After first boot
   of the image you should be able to create the DOS partition
   you also can write a small init script which does that for you
   on first boot automatically
