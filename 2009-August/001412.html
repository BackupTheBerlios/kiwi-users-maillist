> With Kiwi 3.74 on on openSUSE 11.1 x86-64 system I am getting a core 
> dump due to a buffer overflow. It looks like this occurs in 
> /usr/lib/perl5/vendor_perl/5.10.0/x86_64-linux-thread-multi/auto/KIWI/SaT/SaT.so(_wrap_Solver_getInstallList+0x240)
> 
> Stack trace follows:
> 
> *** buffer overflow detected ***: /usr/bin/perl terminated

Install the kiwi version from the 11.1 update repo

http://download.opensuse.org/repositories/Virtualization:/Appliances/openSUSE_update_11.1/i586/

this problem happens when the satsolver is updated. The SaT plugin
in kiwi is linked against the .a lib (there is no satsolver.so) lib
and as soon as your system updates the satsolver stuff the kiwi sat
plugin breaks due to the problem that the sat people don't care
much for abi compatibility

There is already a bug open for it
