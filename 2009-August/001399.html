Hi,

> Is it possible to show some kind of progress while the image is being 
> dumped from the CD/DVD onto a hard drive?
> 
> At present the message just says "Loading image" or something like that. 
> It would be great if progress dots could be displayed while this process 
> is happening.

This is already implemented but requires some tools to be part
in the initrd. The tools are part of the kiwi-tools package but if
the repo you are building the image for doesn't have an updated
version of the kiwi-tools package you only see the message without
the progress. I guess you are building for sle10 so there is no
updated kiwi-tools available, if you build for sle11 you will see
a progress information
