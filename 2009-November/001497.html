Hi,

> I'm trying to create a usbstick  with a single vfat partition that boots of
> linux with a root-fs image file.

I think what you want is a hybrid live system which allows you to
have both a stick with a vfat for data and a linux system with read/write
functionality. If you plan to use openSUSE 11.2 the following should
work for you: http://lizards.opensuse.org/2009/08/05/hybrid-live-systems/

This special iso image contains a partition table and is therefore
exported like a disk to the system. You can run fdisk on the iso
and create your vfat data partition for data and a linux partition
for the OS.
