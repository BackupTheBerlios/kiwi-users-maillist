On Fri, Nov 13, 2009 at 1:08 AM, Eduardo Dominguez <eduardodmz at gmail.com> wrote:
> One of my thin clients, an Acer Aspire L320, stops while booting
>
>  Network module: failed to load network module.
>
> The Aspire L320 is supported by the e1000 driver, which I added to the
> lts.conf file like so:
>
>  MODULE_01=e1000.o
>
> But this doesnt change anything, the clients won't boot.
>
> Am I missing something?
>
Please install the latest kiwi-desc-ltsp, kiwi-ltsp-prebuilt-unstable
and kiwi-ltsp-bootimages-unstable, then run "kiwi-ltsp-setup -c" after
backing up your lts.conf. e1000 module is in the netboot initrd so it
should work, MODULE_01=e1000.o is not required on openSUSE. If you
still cannot boot, select debug option from the boot menu and get us
/var/log/kiwi.log from the client.

Cheers

-J
