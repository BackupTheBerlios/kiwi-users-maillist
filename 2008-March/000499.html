> ok,
> 
> but an update of kiwi will replace thhe config.xml too?

no the contents of your own _system_ image description stays untouched.

the kiwi-desc-* packages provides the boot image descriptions for creating
the initrd which you shouldn't have touched at all. If you had for some
reasons the need for your own boot image description you should have
created a copy of the description directory of your choice and rename it.
In that case an update of the boot image descriptions will not touch
your own stuff
