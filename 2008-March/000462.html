> > If I have a live USB stick, will I be able to use it on any machine or
> > is it for only 1 machine?
> 
> You can use it for any machine. The only problem, as I read somewhere,
> is that the xorg.conf will not automatically be updated. 
> An of course it depends on the architecture. 

yes that's correct. At the moment we don't have code in place which
checks whether the hardware has changed at boot time. This is a problem
for all "static" configurations like for example X with xorg.conf.

As already said the architecture must be protected. So a 32bit x86
stick will work on x86 32bit and 64bit (in 32bit mode) and a 64bit
x86 stick will work on x86 64bit only. Any other architectures for
example ppc will not work because we do not support it at the moment
