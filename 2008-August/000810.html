> I'm trying Minimal LiveCD by using ctso-minimal-suse-11.0
> configuration and produced about 178 MB of iso image. Is this correct
> (belong to size of iso image) ?
>
> Another problem, what is the default user  ? I'm trying linux or root
> with no password with no luck. I also trying to add :
>

You need this in config.sh:

#======================================
# /etc/sudoers hack to fix #297695
# (Installation Live CD: no need to ask for password of root)
#--------------------------------------
sed -i -e "s/ALL ALL=(ALL) ALL/ALL ALL=(ALL) NOPASSWD: ALL/" /etc/sudoers
chmod 0440 /etc/sudoers

/usr/sbin/useradd -m -u 999 linux -c "Live-CD User" -p ""

# delete passwords
passwd -d root
passwd -d linux
# empty password is ok
pam-config -a --nullok
