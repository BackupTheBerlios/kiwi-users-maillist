> According to my project (building KDE 3 Base System LiveCD), I've
> using kde3_basis pattern but the live iso run with init 3 by default.
> 
> What is the missing package / configuration with using kde3_basis
> pattern only ? I want to make GUI LiveCD with  init 5 and automatic
> logon by default ?

to run in init 5 automatically your image description should provide
the file:

	/etc/inittab

just add it as part of the overlay directory of your image description.
The overlay directory is the one named root/. Everything below root is
copied as it is. So if root/etc/inittab exists it will overwrite the
default one.

Your inittab copy should change the following line:

	id:3:initdefault:

to

	id:5:initdefault:

that's it

I knew by design of kde and gnome they are big boys :-) Maybe you
