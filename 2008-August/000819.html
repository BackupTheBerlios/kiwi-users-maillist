The problem turned out to be that my Qt libs are not installed from a  
rpm Package, but from tar file in a non standard location. The lines
     $QTLIBS = `pkg-config --libs QtCore`;
     $QTINC = `pkg-config --cflags QtCore`;
in tools/dbuslock/Makefile.PL failed to find the required information.  
HOWEVER, the script did not fail at this point but happily write a  
Makefile ... so the error went unnoticed.

I have created a "fake" package description and put it into  
/usr/lib/pkgconfig. Now it builds and installs, kiwi starts, but fails  
later with something else ...
