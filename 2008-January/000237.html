The boot image (initrd) and the system image includes different
kernel versions. It looks like your PXE environment provides outdated
boot or system images. Please note if you create the pxe image type
kiwi automatically creates the system and the boot image which always
fits together but you need to make sure this data is present on
the PXE boot server
