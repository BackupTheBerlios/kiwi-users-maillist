Ah, Ralf tftp is controlled by init atftp is a daemon started
by rc script. Just install atftp again and edit the file
/etc/sysconfig/atftpd: Set the following options there:

	ATFTPD_OPTIONS="--daemon --no-multicast"
	ATFTPD_DIRECTORY="/srv/tftpboot"

now make sure there is no old tftp running:

	init q
	ps ax | grep tftp --> kill them

now start atftp

	rcatftpd start 

you really should use atftp
