I think here is the problem, why do you need to modify the isoboot
config.xml ? If you think you need to do this because of the repo's
it's not true kiwi automatically adapts the repo according to your
system (test) image description. Please don't change anything in the
kiwi boot images.

I think you should get a working iso if you do the following:

1) re-install all kiwi packages
2) change the test1 config.xml according to your local repo
3) run kiwi prepare, create
