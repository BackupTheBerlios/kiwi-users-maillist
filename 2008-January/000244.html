so just to make sure:

- you copied the initrd and kernel files to /srv/tftpboot/boot/ ?
- you configured PXE to use this files by correct links (linux,initrd) ?
- you can see on the boot screen of the client that the PXE Bios
  picks up this linux and initrd files ?
- you copied the system image and md5sum file to /srv/tftpboot/image ?
- your config.<MAC> in /srv/tftpboot/KIWI/ points to the correct
  system image located in /srv/tftpboot/image ?
 
> What can I do to check about versions etc.?

the kernel version is part of the initrd* file name. If you want to
check if this kernel is also part of your system image you can loop
mount the image file and check if /lib/modules/* includes the kernel.
But I'm sure if one kiwi create process creates all the files in
/tmp/mystem-result they are all based on the same kernel
