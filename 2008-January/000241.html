Hi,

> How do I control the display size for the kwliveCD vmware image?  I'd
> like to default to 1024x768.

you mean you want that resolution to be set automatically ?
if yes you can write a small sax2 batch script. Just put in a file
for example:

	Screen > [X] -> Depth-> 16 -> Modes  = "1024x768"

this will enable 1024x768 for the 16bit color depth
sax2 is called as part of the service:

	/etc/init.d/create_xconf

if you change the call to be

	sax2 -c 0 -a -b <your-batch-script>

it should activate the resolution you want
