Hallo,

it was a hard and long fight but now my pxe/nfs Testsystem in vmware is
working now.

Some hints

1)
Vmware is using an own dhcp-server von nat and host-only.
This could cause problems with your own dhcp-server

2)
Check your nfs-server in three steps
a)
I nfs-mount working local?
b)
Is nfs-mount working remote from a client that works perfect with
another nfs-server?
c)
Is nfs working after the pxe-image is started?
If /mnt contains nothing or the imagefile something is wrong.
verify the /etc/export example on page 23 of the openSUSE-Kiwi Image
system file with your own configuration.

3)
Using external installation-sources could be a problem when using
vmware. This depends on your own netzwerk-configuration.
I m using a local mirror on the nfs-server.

4)
Never use a pxe/tftp/dhcp-server in a foreign network without asking the
administrator. Check this very carefully.
