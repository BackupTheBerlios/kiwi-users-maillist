This is the first step to install openSUSE 11.0 on this netbook.

The installation goes over LAN because the WLAN-Module isn't
supported of the kernel yet and I miss a configuration tool for
WLAN while the installation of openSUSE. Can this tool integrated on
the NET-Install-CD? This is a good idea for further opensuse versions
to install opensuse over WLAN.

After the installation, I compiled the WLAN Kernel-Module from the
source. I'm very glad, that the driver supported directly WPA1 and WPA2.
It's running without any problems. This is great.

The Akoya Mini E1210 runs with openSUSE 11.0 now and I don't give
this netbook away. *gg* This is the one of netbooks that you can
use reasonably ten-finger-system. All others have undersized keys.
