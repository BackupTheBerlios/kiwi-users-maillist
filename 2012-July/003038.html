Hi Anderson, 

If I recall, OpenSUSE does not enable SELinux by default because it has
been problematic in the past and does require a host of tweaks to ensure
things work as expected. 

On 12.1 there was a libselinux1 package that was installed, but it
simply contains a single shared library, so disabling selinux at the
kernel level should disable it in the image.

I also believe there is a command called sestatus that will allow you to
test if the selinux is enabled.
