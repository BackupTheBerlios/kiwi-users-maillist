looks like a broken kiwi installation. Can you check with

  rpm -Va

if your installation is valid ?

the perl bindings are provided by the kiwi base package
so "rpm -ql kiwi" should show you something like

  /usr/lib/perl5/vendor_perl/5.10.1/x86_64-linux-thread-multi/KIWI/dbusdevice.pm

maybe your perl version doesn't correspond with the kiwi perl
bindings but this should have been reported by rpm on install
