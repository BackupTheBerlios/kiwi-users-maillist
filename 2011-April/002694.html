The "new version" of creating USB sticks was released with 4.81. Prior 
to 4.81 you should have an <type> specification that looks similar to 
the following

<type image="usb" filesystem="...." boot="usbboot/......"/>

and then run prepare and build.

The description you pointed out in your previous e-mail was changed in 
release 4.81 or later. It appears that maybe your packages are not in 
sync, doc package maybe ahead of the rest of kiwi?

Yet the error you described is consistent with the example of 
suse-live-stick for version 4.43, as in 4.43 the example did not have a 
<type image="oem"...../> specification, but in version 4.81 or later it 
does.

Something doesn't quite match.
