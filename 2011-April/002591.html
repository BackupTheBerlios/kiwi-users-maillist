The aufs filesystem is needed when you set the "flags" attribute on the 
<type> element. If you do not use the "Flags" attribute then "aufs" is 
not needed. The disadvantage is that you will get less data into your 
image as the image is not compressed.

aufs is not supported by SLES, and since you are working on ia64 you 
will not be able to get a package from the build service either. Thus 
you would have to build your own package on your machine.

>
> LSB_VERSION="core-2.0-noarch:core-3.2-noarch:core-4.0-noarch:core-2.0-ia64:core-3.2-ia64:core-4.0-ia64"
>
> SUSE Linux Enterprise Server 11 (ia64)

Note Kiwi does not provide any boot descriptions for ia64 and we do not 
officially support or test this architecture. Also non of the startup 
scripts have been ported to the ia64 architecture. You will need to 
create your own boot description and scripts as needed. Although 
everything else should mostly work there might be a few places where you 
might have to change Kiwi code.
