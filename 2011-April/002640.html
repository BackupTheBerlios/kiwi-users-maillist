Hello,

I am attempting to create a SLES11-SP1 OEM appliance image which is installed on a
LVM root partition.   The kiwi prep and create process completes successfully.

The ISO install also completes successfully.   However, upon rebooting the appliance
the root fails to be mounted, with the following message on the console:

	"Waiting for device /dev/pfc8V/LVRoot to appear".

At first blush it appeared that the installed initrd was missing ext2 support for the
/boot partition - I added this using the <drivers> element but that did not fix the issue so
there appears to be other features missing from the initrd.

TIA,

- john
