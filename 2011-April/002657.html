> We want to use OEM kiwi builds to both install via ISO/CDR, and via network (PXE).  We would also like to use LVM partitioning.  We have tried a few builds with LVM configured and it doesn't look like the PXE setup is handling the image file correctly when it is an LVM build.  Should this work?
> 
> (sorry, don't have exact error messages available right now)

when deploying an oem LVM image via pxe it's important that you
add the paramter: 

   VGROUP=...

to your pxe configuration append line. By default the volume group
name is kiwiVG, so this means VGROUP=kiwiVG. If you have choosen
another name for the volume group in your system image config.XML
you have to add that name respectively

If it still doesn't work I need the /var/log/boot.kiwi file
(boot with kiwidebug=1 in the pxe append line)
