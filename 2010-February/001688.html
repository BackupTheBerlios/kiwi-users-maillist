Hi,

> I'm working on a liveCD to create a web service appliance.
> I need the liveCD to boot for itself but also to make a persistent installation on some disk.
> What is the right way? And, how to configure a custom grub's menu.lst from KIWI?

I would say create the live CD with kiwi and put the YaST2 live installer
into your live CD image. The yast2 live installer allows you to install
the running live system including its current configuration onto a disk

Another solution would be to create two images. One which is of type iso
and represents your live CD and another which is of type oem and
represents the same image as installable media.

   <type image="iso" flags="clic" boot="isoboot/suse-11.2"/>
   <type image="oem" filesystem="ext3" boot="oemboot/suse-11.2" format="iso">

   kiwi --prepare ...
   kiwi --create ... --type iso  => your live CD
   kiwi --create ... --type oem  => your install CD
