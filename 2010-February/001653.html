Hi,

> Everything works perfect until now. If I try to boot the ISO Image, I
> get a Kernel Error, that the Root Filesystem cannot be mounted. (see
> Image attached)
> 
> Any ideas how to fix this issue? Where is my fault?

Problem is that the type in config.xml is set to

   <type image="iso" boot="isoboot/suse-SLES11" flags="unified"></type>

which requires aufs to be present in the target system. But
aufs does not exist in SLE11 so the boot fails. You can solve this by
either using 11.1 as base distro or change flags to "compressed"
instead of "unified"
