> My question is, given that the type is specified in the config.xml file,
> does the command line --type add any useful value? 

The --type command line argument is used to switch between different 
image types in they are configured in config.xml. For example in a 
config.xml file with

<type image="vmx" filesystem="ext3" boot="vmxboot/suse-11.0"/>
<type image="xen" filesystem="ext3" boot="xenboot/suse-11.0"/>

One can switch between creating VMware image of a Xen image by using the 
--type command line argument.

If the pxe image is the only one you have configured in config.xml then 
Kiwi will use it as the default and generate the PXE image, thus --type 
is not required on the command line.
