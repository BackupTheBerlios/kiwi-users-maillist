creates a kiwi image description as a result. So if the system you
are migrating has mysql installed the kiwi migration process should
have done one of the following actions:

a) it could find mysql in one of the repos part of the zypper repo list
   it will add this package to the packages list except it is part of
   an installed pattern. If it is part of a pattern the pattern will
   be added.

b) it can't find mysql in any repo defined by zypper. it will report
   this as a problem in the report file
   
So what applies in your case a) or b) It would be also helpful if
