NOTE:
-----
as you are about to build a SLE10 image be aware if you are doing
that on a non SLE10 build machine but on a newer suse version like
11.0 or 11.1 make sure you pass the following option to kiwi when 
you create the image

   kiwi --create .... --fs-inodesize 128

the default inodesize has changed in the filesystem tools on newer
distro versions. If you use a newer filesystem tool to create the
filesystem but the SLE10 grub bootloader it will not be able to
read the filesystem because of the different inode sizes
