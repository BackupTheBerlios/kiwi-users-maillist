oh, yes the default password for the root user in the template
is simply "linux" :-) You should do the following in your config.xml
to make your own users:

    <users group="root">
        <user pwd="crypted-password-string" home="/root" name="root"/>
    </users>

and remove the passwd copy inside the image description

    root/etc/passwd

The "crypted-password-string" can be created with kiwi itself

    kiwi --createpassword
