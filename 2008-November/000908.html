Hi,

> This is just for your information and for all people who use
> smart and openSUSE 11.0 with kiwi
> 
> The kiwi packages built on BS for 11.0 use a newer version
> of libsatsolver-devel and not the original package from the
> 11.0 distribution to build the kiwi packages. The result is
> that kiwi complains about:
> 
>   unsupported SOLV version
> 
> The problem ist that the solvable file is created with the
> 11.0 sat tools but the result can't be read because the
> packages have been built with a newer library version of
> the satsolver. I already get in contact with the buildservice
> people to find the reason of the problem but didn't get
> a response so far
> 
> If you have that issue you can solve it as follows:
> 
>   - make sure your system has rpm-devel, a C compiler and
>     swig installed
> 
>   - make sure you have installed the libsatsolver-devel
>     package from your 11.0 distro (DVD)
> 
>   - download the kiwi sources:
>     svn co https://svn.berlios.de/svnroot/repos/kiwi/kiwi-head .
> 
>   - cd tools/satplugin
>     make -f Makefile.swig
>     make
>     sudo make install

This issue is fixed, it was a schaefi too stupid to build
correct packages bug
