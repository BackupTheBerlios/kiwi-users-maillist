Hi,

> I'm trying to deploy a image to two different usb sticks, but on both it stays forever in "Creating partition table on: <device>".

Hmm, how big is the cb-live-stick.i686-0.2.1 file compared to the size
of the stick ? Do you use a squashfs compressed stick ? did you check
if the stick wasn't mounted by any other program before calling
kiwi --bootstick ?

What kiwi does is it calls fdisk and after that it checks the size of
the created partitions. If the size is ok for the image it puts it on
the stick if not it increases the partition size and restarts fdisk
again. kiwi does that because of the alignment done by fdisk. I have
also implemented support for parted but didn't test it hard enough.
If you like you can give it a try with the option:

   --partitioner parted
