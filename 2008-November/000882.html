>      But while using that image its asking for username and password. 
>      i tried many options, but did not go through it. 
>      did not find anything in config.xml related to users. 
>      can u please tell me which username and password work for it? 
>      or how to reset the password for root. 

oh, yes the default password for the root user in the template
is simply "linux" :-) You should do the following in your config.xml
to make your own users:

    <users group="root">
        <user pwd="crypted-password-string" home="/root" name="root"/>
    </users>

and remove the passwd copy inside the image description

    root/etc/passwd

The "crypted-password-string" can be created with kiwi itself

    kiwi --createpassword
