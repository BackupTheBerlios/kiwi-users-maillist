On 04/24/2012 01:20 PM, balaguru wrote:
> hi,
> I am very new to kiwi build system and don't know how to achieve this.We
> are trying to create vmware appliance(ovf) with multiple harddisks,one
> contains system files(softwares,system files,etc.) and another contains
> just data.Idea is in case of software updates just replace primary disk
> which contains system files and retain data disk.Something like this we
> want to achieve in kiwi. We kind of stuck up here.If anyone could help
> I'd really appreciate.

KIWI does not directly support this. Then again this is very easy to 
script or follow through manually.

All you have to do is create an image file:

qemu-img create -f vmdk [ANY_OPTIONS] myDatadisk.vmdk [SIZE]

You can loopmount the image file to format the disk image.

In the .vmx file you just need to add the path to your file as an 
additional disk entry.

done.
