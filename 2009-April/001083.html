Hi Robert,

> For learning purposes I've created a configuration that produces an image 
> with autologin (in console mode, i.e. init3) and X server startup. This 
> works nicely on 32 bit when testing with qemu -cdrom image.iso, but with a 
> 64 bit image on x86-64 I am running into X server issue:
>
> qemu-kvm -cdrom image.iso
>
> produces the output shown in the attachement. Any help to address this is 
> appreciated.

The problem is that you don't have a module path for 64bit set
in your xorg.conf. You need to have it like that

  ModulePath    "/usr/X11R6/lib/modules"
  ModulePath    "/usr/X11R6/lib64/modules"
