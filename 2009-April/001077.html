Hi,

> I am building a live usb stick image ( split image with ext3 and
> squashfs ) and want to explicitly determine the size of / and /home,
> say, 1.5G and 1G.
> Is there any way to do that? I guess I need to modify the suse-repart
> script or some other kiwi modules?

you can do that by calling

  du -s --block-size=1

kiwi does that in KIWIBoot.pm. I don't know when and for what you
need that size values but you can use the config.sh or images.sh
scripts to run the "du" call
