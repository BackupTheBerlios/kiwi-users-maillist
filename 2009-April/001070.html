> Hi,
>
>> How do I get rid of the grub menu in an image created with KIWI.
>> Basically I'd like to see the welcome screen and then go directly to
>> booting the kernel.
>
> I'm sorry there is always a menu by default as you should be able
> to select a failsafe boot entry.
>
> Talking about a virtual disk you can change the grub setup after
> the image has been created as follows:
>
>   losetup /dev/loop0 <name-of-raw-file>
>   kpartx -a /dev/loop0
>
>   mount /dev/mapper/loop0p1 /mnt
>
>   ===> vi /mnt/boot/grub/menu.lst
>
>   umount /mnt
>   kpartx  -d /dev/loop0
>   losetup -d /dev/loop0
>
