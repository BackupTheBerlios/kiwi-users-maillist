The problem is that the bootkernel specifies the name of a a profile
_not_ the name of a kernel package. So if you have a look at the
config.xml file in the corresponding kiwi boot image description
in /usr/share/kiwi/image you will find something like

   <packages type="image" profiles="pae">
        <package name="kernel-pae"/>
        <package name="kernel-pae-base"/>
        <package name="kernel-pae-extra"/>
        <package name="aufs-kmp-pae"/>
   </packages>

So if you do the same change there it will work. But better is to
copy the entire boot image description and do the change there
and reference this new boot image description in the boot="..."
attribute in your system image config.xml
