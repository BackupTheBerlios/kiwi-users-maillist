You are right there is no difference in:

   <type boot="oemboot/suse-10.2" filesystem="ext3" format="vmdk">vmx</type>
   kiwi --installcd .../initrd-oemboot-... --installcd-system ...

and

   <type boot="oemboot/suse-10.2" filesystem="ext3" format="iso">vmx</type>

The format="iso" causes the system to call exactly the same code compared
to kiwi --installcd so there shouldn't be any difference.
