I have added my own pattern handling in kiwi because except zypper
no other package manager can handle it. zypper does some special
exception which causes packages _not_ to be installed. For now I
don't know if that happens intentionally or if it's a bug in zypper :)
Anyway you get a clean and working system using smart in combination
with opensusePattern

To find out what packages makes a pattern you only can have a look
into the file itself. Right now kiwi does not have a dry-run option
or something which just would create an information log. Don't
know if this would be a good idea ?
