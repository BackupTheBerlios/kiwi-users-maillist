On 9/18/07, Mark Dickie <markd at hotdesk.biz> wrote:
> Hi,
>
> Just to confirm I am putting a 10.3 i386 DVD in the drive while running the
> kiwi-ltsp-setup

You would have to change the installation source in
/etc/sysconfig/kiwi-ltsp as /media/10.3mountpoint, use the one created
by the DVD.

Make sure you are using the latest smart rpm as mentioned on
http://en.opensuse.org/LTSP.

After that run "kiwi-ltsp-setup -p" this will run prerequisite check.
If that is successful, run "kiwi-ltsp-setup -s".

Please make sure all the required variables are entered correctly in
/etc/sysconfig/kiwi-ltsp.
