On Tue, 2007-09-04 at 15:47 +0200, Marcus Schäfer wrote:
> Hi All,
> 
> v1.60 is building. A very nice change is the new web-page logo
> on http://kiwi.berlios.de. This one was sponsored by 
> biswajyotim mahanta <biswajyotim at gmail.com>. I like it :-) 
> The following is a list of bugfixes and enhancements:
> 
> Enhancements:
> =============
>     - added support for <packages type="delete"> section (#304765)
>       you can add an additional section named:
> 
>       <packages type="delete">
>           <package name="...">
>           ...
>       </packages>
>      
>       in any kiwi script you have the baseGetPackagesForDeletion()
>       function available which can be used to remove packages. For
>       example
> 
>       for package_name in `baseGetPackagesForDeletion`;do
>          ....
>       done
> 
>       I have adapted all *boot description to make use of
>       this feature
> 
>     - Allow remote root/ to be exported via network block device.
>       If a server has a valid nbd-server setup the kiwi config.MAC
>       file can use this root tree by setting up the following line:
> 
>           NBDROOT=ip.of.server.machine;Port;/dev/nbd0
> 
>       The last information is the device name exported by the
>       server. This is an option and by default set to /dev/nbd0
>       We still have some problems with this feature, so any test
>       results are highly appreciated
> 
> Other Bugfixes:
> ===============
>     - fixed network block device default name: is now nbd0
>     - allow block device name to be specified in NBDROOT
>     - added patch by Jan <jcbornschlegel at novell.com> which fixes
>       directory changes not happen in subshells
>     - removed isoinstboot description from repo, not valid anymore
>     - got a nice cover page by Frank Sundermeyer <fs at novell.com> for
>       the documentation, thanks Frank
>    
> Have Fun :-)
>  
> Regards,
> Marcus
