Hi,

> is there any way to change the default gfxboot 800x600 resolution in
> case of an iso-boot (isoboot/suse-11.3)?

as I wrote the vga="..." attribute should do the trick, if not
there is a bug which needs a fix :)

> Or is there any way to switch off the graphical boot screen completely?

you can swich off boot graphics by just not installing the
gfxboot branding packages. In your config.xml you will find
something like:

   <package name="gfxboot-branding-openSUSE" ...../>

simply remove that line and there is no graphics boot anymore
