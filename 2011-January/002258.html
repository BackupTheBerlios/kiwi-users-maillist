On 01/17/2011 04:18 PM, john.lund at emc.com wrote:
> We have been using a kiwi config.xml that has multiple profiles and would like to continue doing this.  We have been trying out the 4.70 build, and are getting the following error:
> Found reference to profile  AtmosDevelop but this profile does not exist.
> I don't think we are doing anything not allowed in our config.xml.  Just to be sure I checked our config file via jing against the KIWISchema.rng and I don't get any error messages back.  Are there any known issues with multiple profiles and 4.70?

This error is is triggered by the consistency checker and thus would not
be found via jing. Basically the config.xml file contains an element
that has a profiles attribute

<....profiles="AtmosDevelop>

but the config.xml file is missing a

<profile name="AtmosDevelop">

element.
