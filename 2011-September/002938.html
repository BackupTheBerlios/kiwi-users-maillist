Autologin:
there are two solutions that I use:

1. image built with kiwi
we do use some specific startup stuff, which amounts to starting
a script /etc/init.d/startX
in which
   start-stop-daemon ... -u otheruser  ... runX
is called. in runX there is a line
   xinit /usr/bin/fvwm -- :0 -br vt7 ...
which will make X started as "otheruser" and everything will be fine.
This additional stuff is put into several rpm packages, which are
installed into the kiwi made image via config.xml.

2. openmoko phone with Debian
there is a small utility called "nodm" which will start X as
user otheruser if so instructed, and all X clients need to show up
in otheruser's home dir in .xinit.

maybe some of this helps.
