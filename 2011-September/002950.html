> In a lot of cases someone who focusses on ARM tablet etc. development
> won't have an ARM server around. So I wonder won't it be an option to

user mode linux should also work in the same way as a virtual machine
solution. OBS uses VM's based on Xen or KVM to build packages and images.
So yes that environment would be perfectly fine to build the images.
for ppc we should be able to do that today because kiwi already supports
building of ppc images. Yury Tsarev <ytsarev at suse.de> added the support
for it. He builds on ppc based blade servers. For arm we still have
the problem that I don't have an rpm based distro at hand. So if I'd
like to do it together with Debian we need to hack kiwi for dpgk and
the boot code. I definitly could need help here
 
> I wouldn't mind waiting longer, to have the image build.

  https://build.opensuse.org/project/show?project=openSUSE%3AFactory%3AARM
