Hi,

> Trying a template (suse-11.4-JeOS) with PXE configuration does not
> get enough ramdisk for the image:
> 
> http://i39.tinypic.com/t0kh1c.png
> 
> How can I increase the size of /dev/ram1? The guest has 1500MB of
> ram, so there should be enough space.

use the following kernel boot option 

   ramdisk_size=800000

that's ~800M for each ramdisk. I have that option set in my
pxelinux /srv/tftpboot/pxelinux.cfg/default config
