For your testings you can easily use a self written script which does
the rpm database conversion. The script can be set as parameter to
the kiwi option:

	--prechroot-call

The given script is called before kiwi starts operating in the
chroot. The first parameter of the script is the location of the
root tree. The script itself does _not_ run chrooted

if it works for you I can patch kiwi directly according to your
findings
