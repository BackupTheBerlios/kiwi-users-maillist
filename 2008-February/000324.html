Sorry one thing I forgot. If you use the flags="compressed" variant
for live systems there is a change. In my last announcement I
wrote that we switched to the new split system in this case.
If you use it too you need to add a split section in your system
image config.xml which controls the read-write data. A good example
to start with is:

  <split>
    <temporary>
      <!-- allow read/write access to: -->
      <file name="/var"/>
      <file name="/var/*"/>
      <file name="/etc"/>
      <file name="/etc/*"/>
    </temporary>
  </split>

See also the kiwi-devel thread with the topic:

	[PATCH] enhanced split image support in Live CD
