Hi,

> Are there compelling reasons not to have swap space in a vm image?

I thought it wouldn't make much sense as you can control the
amount of RAM in a VM yourself. Additionally the swap would happen
into a virtual disk which means the virtual IO as well as the real
IO layer would be used to swap virtual RAM data into a file.
I think it would decrease the performance of the system a lot
but I'm not 100% sure if this is really the case
