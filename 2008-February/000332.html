> 1. The size tag is ignored - regardless of what i set there, the image will always have a size of ~600MB.

the problem is you are creating a virtual disk and this disk is created
as a sparse file. the size information in the config.xml would apply if you
are creating a filesystem based only image. So to influence the size
in your case you need to influence the size of the virtual disk. This
can be done by setting the paramater
 
	--bootvm-disksize 2GB

while you call kiwi with --create
