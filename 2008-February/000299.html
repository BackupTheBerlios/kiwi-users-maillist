I have been playing with KIWI for a week or so. My target is to make a
minimal diskless boot using NBD.  It system will run in level 3, so
there will be no X. For the most part I think I know what is going on.
Except for:

1) If I do not specify a kernel in my config.xml file, I still get a set
of images. However, after the kernel boots and the nbd image is
