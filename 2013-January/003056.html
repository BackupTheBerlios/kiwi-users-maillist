It's not required to change kiwi here because this is already
supported:

   kiwi --prepare ... --root /existing/root --recycle-root

> Also, due to disk space constraints, I had to
>  * add "--download as-needed" to
>    the zypper command line of kiwi.
> With these settings, the kiwi perpare step is able
> to install a working system directly on hard disk.

I understand but unfortunately I don't like to change

to the installation. I would be fine with a patch which
allows that as a commandline option ;)
