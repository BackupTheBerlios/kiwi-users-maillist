Changes Report kiwi v4.61 2010-09-26 - 2010-10-01
--------------------------------------------------
most important changes for last weeks release...

commit 88495c145a7f84419e5b0b7b58054ce77d2785cd

    - fixed the creation of the install CD if the syslinux
      bootloader is used.

commit 8ba200c17cc18d016e5decfb77880dec790ba80c
commit bdd4aa02ad9801e4063931dca643188fb32185ab

    - fixed OEMRepartStandard() to use the suggested last cylinder for
      the last partition to become created instead of a fixed value.
      Also use the suggested last cylinder for the resize of the root
      partition if there is no other partition to become created and
      no oem-systemsize was specified. Thus leave it up to the partition
      tool to find the end of the disk
