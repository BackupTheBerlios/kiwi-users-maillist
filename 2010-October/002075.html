Hi Axel,

On 10/07/2010 07:11 AM, Axel Schmidt wrote:
> Hi Marcus,
> 
> I am actual using kiwi in version v3.01 from SLEPOS11 and I am trying to
> include "/usr/share/doc/packages/cups" in the created Kiwi image.
> 
> It seems that all rpm'a are always installed using the the
> "--excludedocs" option regardless if
> 
>  <rpm-excludedocs>False</rpm-excludedocs> is set to False or True
> checking the result after running "kiwi -p"
> 
> Is there any way how to do it?

The removal probably happens in config.sh. Some of the examples from the
kiwi-doc package have the following code in config.sh

rm -rf /usr/share/doc/packages/*
rm -rf /usr/share/doc/manual/*

If you started your configuration tree via copy of an example this is
probably the culprit.
