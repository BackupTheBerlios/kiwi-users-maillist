I'm pretty sure this is a problem with clicfs. As you already know
the libs are part of your unpacked tree it must be a problem when
mounting the clicfs packed tree. This could be a fuse, kernel, clicfs
problem.

1) if you change back to the 11.4 update kernel-desktop (just deactivate
   the Tumbleweed repo and rebuild). Does the problem disappear ?
   
2) if yes, try to update the clicfs package on your image build machine
   with the version from here:

   http://download.opensuse.org/repositories/filesystems/openSUSE_11.4/

3) after that make sure you use the same clicfs version as part of your
   image. So add another repo entry as follows

   <repository type="rpm-md">
      <source path="http://download.opensuse.org/repositories/filesystems/openSUSE_11.4"/>
   </repository>

4) now activate the Tumbleweed repo again and rebuild the entire
   image. Any difference ?

Hope that helps
