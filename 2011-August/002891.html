Hi,

> Not sure if this is is linked to Kiwi or OpenSUSE but after each build the
> default keyboard layout is changed to German. Below are the relevant
> settings in the config file, has anyone else run into this behavior before?
> 
> <locale>en_US</locale>
> <keytable>us.map.gz</keytable>
> <timezone>US/Eastern</timezone>

Hmm, that's strange. Do you have use overlay files like
etc/sysconfig/keyboard in your image description ? Do you run
yast or other configuration tools on first boot of the system

The keytable from config.xml is evaluated if the function
suseConfig() runs. This is normally done in config.sh. In other
words the setup of the keytable normally happens at the end
of the kiwi prepare step. So you can check if the keytable
is setup correctly after your kiwi prepare step is done. If
it's still ok at that stage it must be something which runs
on boot of the image

Hope that helps
