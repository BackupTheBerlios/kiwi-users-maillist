Hello there,

I recently built an other openSUSE image with KIWI and noticed a strange
thing. I've installed man, but I can not view any man pages:

linux:~ # man man
No manual entry for man

So I quickly checked if man is installed or what?

linux:~ # zypper if man
...
Installed: Yes
Status: up-to-date
Installed Size: 1,8 MiB
...

linux:~ # zypper in man
'man' is already installed.
No update candidate for 'man-2.5.2-43.2.2.i586'. The highest available
version is already installed.

Seems fine. The first idea was to reinstall the package:

linux:~ # zypper in --force man
...

And voila, we have man pages:

linux:~ # man man | wc
    599    4034   30740

Why is that? See the attached config.xml for the image configuration.

Thank you,
