yes you really need the packages marked as recommended when building
a desktop appliance. The pattern system with regard to desktops doesn't
really work when installing only the required packages. The system
integrity is ok but the desktop really "requires" more which is indicated
as "recommended". In my opinion a big mistake of the design of patterns.
If I state I want to have the kde pattern installed than I also would
expect it to work but this is only the case if also the recommended
packages are installed which is not obvious
