Solved.

1. Disk must be prepartitioned and formatted to known filesystem, so there
is enough space for each partition in config.<MAC>

2. No "x" in size field of config.<MAC>: only precise numbers that are
slightly more than enough to contain the image.

3. Mount point should exist (just mkdir /tmp/mypxe/mountpoint after
"prepare" kiwi step but befote "create" kiwi step)

I'm not quite sure if all this points mandatory and/or understandable, this
is some kind of "it just works" magic.
