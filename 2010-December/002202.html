Changes Report kiwi v4.68 2010-12-05 - 2010-12-10
--------------------------------------------------
most important changes for this weeks release...

commit db045f795f9329807a301f1235547c490020867f

    - fixed location of local modprobe configuration file.
      The new place is in /etc/modprobe.d/99-local.conf. Only if this
      doesn't exist we will use /etc/modprobe.conf.local

 in order to have this fix completed I had to checkin
 another fix which didn't make it into this release. If you urgently
 need it before next weeks relase just drop me a note or patch
 your local version with the commit:

     cb1e3bf223394b603adef01b4b52f54981fc8c6f


commit 3514d5d0652305762c1fa42b91b343bbc5e10e2e

    - make baseSetupInPlaceGITRepository more useful

 I'm working on a possibility to track changes during the lifetime
 of an appliance before it will be updated. I'm currently playing
 around with git as I think it's a good tool to do that. You are
 all invited to share even the most crazy idea about this topic
 with us :-)


commit 6d3ab89c806a03e807f5cafd5b5aec2e6e6cb74f

    - added Option --check-config which validates the XML using the
      new KIWIXMLValidator module. Along with the new module validation
      from KIWIXML has been moved into the KIWIXMLValidator object

 You can call:

    kiwi --check-config /path/to/config.xml and it will check
    if the configuration is ok. Thanks to Robert for doing this


commit bb6d909aff8c0dc81ba44846270363ab3fe447b1

    - added updateTypeList() function in KIWIXML which is required
      if an accessor function changes the XML tree the internal data
      structures needs to be updated too

    most important fix for this release as it caused several
    regressions :( Hope you didn't had too many problems with it


Remember to have fun...
