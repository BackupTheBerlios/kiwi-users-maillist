Dear Marcus,

thank you for the reply. Actually the problem seems to be with this:

> cat /etc/modprobe.conf.local
> options brd rd_size=512000

If I put this line to the file /etc/modprobe.d/99-local.conf, then 
everything works as expected.

There is a hint for this behaviour in  "man modprobe.conf":

"the /etc/modprobe.conf file can also be used if it exists, but that will 
be removed in a future version"
