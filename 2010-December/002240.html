> Hi,
>
>> Or all my partitions (except image partition) are safe and in case of
>> problems kiwi just stops and will not corrupt any existing non-root
>> partition by repartition?
>>
>> E.g.
>>
>> PART=5;S;x,200;L;/,100;L;/mountpoint,
>>
>> Can I not to bother about safety of data on sda3 during some repartition
>> after pxeboot?
>
> all data on the disk specified by DISK=... will be lost because
> kiwi creates a new partition table and wipes the existing one
> in this imaging mode. if you want to have some parts on the disk
> not touched it might be better to use kiwi just to create the root
> filesystem image and write your own boot code as clone from netboot
> suitable to deploy this image on your client machines.
