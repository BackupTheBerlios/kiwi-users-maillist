ok, so squashfs + aufs created the correct images but failed to
mount. I think this is because of an incompatibility between the
building host mksquashfs and the kernel mounting the filesystem
later on. You can check that if you boot with: 

   kiwidebug=1

and check the kernel messages (dmesg) it tells you about
squashfs version conflicts if my assumption is correct.

I'm now going to reproduce the checksum and other errors

worked fine for me with latest kiwi and a 11.2 build host
