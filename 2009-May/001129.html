> Currently i have to migrate my imaging machine from the Dell E4200 to a
> Optiplex 755. I installed the same distribution and the same version of
> kiwi, the creation process is not a problem everything work well then I burn

Hmm, what type of keyboard is this, usb ? It's hard to debug
something like this. What you can do is connect a standard usb
keyboard to the laptop and try again. kiwi loads the usbhid
module and therefore each usb keyboard should work

if you have a working setup you can pass the
kernel parameter kiwidebug=1 when the question comes up
you will have a shell at Alt-F2 and you can do some
debugging why the internal keyboard doesn't work
