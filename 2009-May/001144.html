Hi Jose,

> I'm using Kiwi to build a VMware-based image with SLED 11 instead of OpenSUSE.
> I've borrowed the config.xml provided under /usr/share/doc/packages/kiwi/examples/suse-11.1/suse-vm-guest and I was able to run the first stage (kiwi -p) with no issues (after making the necessary changes to the names of the pattern and bootsplash).

I think the easiest way to get you started is if you install/update
your kiwi to the latest version from here first:

 http://ftp5.gwdg.de/pub/opensuse/repositories/Virtualization:/Appliances/SLE_11/

after that make sure you have installed the kiwi-templates package
This package provides SLE11 examples which you can see by typing

   kiwi -l

To build and create the template in one step call

   mkdir -p /tmp/myimage
   kiwi --build suse-SLE11-JeOS -d /tmp/myimage \
        --set-repo /path/to/sle11/repo
