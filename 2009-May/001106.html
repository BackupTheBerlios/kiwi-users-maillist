Hi,

> my target system has problems with the default resolution so I want to 
> change the vga=0xXYZ param.

ok, I will add that to the iso type too. We have that already
for the virtual disk types. the vga value is passed as vga
attribute as part of the type setup:

    <type ... vga="...">iso</type>

> Furthermore a active serial konsole to watch the boot progress would be 
> fine.

for live systems I can generally add the following: 

    console=ttyS0,9600n8 console=tty0

you will have both, an active serial console and the
standard system console at boot time. For serial console
access you need to have /etc/securetty as part of your
image

Would that solve your request ?
