Changes Report <kiwi 4.59> 2010-09-05 - 2010-09-10
--------------------------------------------------
The last two weeks we had lots of changes and commits,
therefore I will not reference to the ID's but write
about the changes in a short description. Don't hesitate
to ask for details if one or the other change falls into
your playground. 

most important changes for this release are:

- The documentation has been moved from LaTeX to DocBook_4
  along with this change several doc fixes were also made
  We now provide a PDF and a HTML documentation for kiwi
  with the kiwi-doc package. Of course the manual pages
  are also created as extra man pages. Special Thanks
  to Thomas Schraitle who is really the DocBook master :-)

- kiwi now supports the s390 target with the image types
  vmx and oem. It's possible to create images for the
  DASD and Virtio (KVM) disks. The SCSI disk (zFCP) will
  follow soon. The repartition code for DASD devices is
  still work in progress and is planned for the next week
  release

- Special thanks to Chris Bowlby who added the PXE MAC
  grouping feature. There is also a documentation section in the
  PXE chapter if you are interested in more information

- Along with the s390 support the kiwi code received some
  cleanup. Especially in KIWIBoot.pm some code parts moved
  into functions and many duplicate code parts were
  removed
  
Regards
Marcus
