Hi John,

According to the zypper developers you do not need to add the 2nd DVD
ISO file as a repository as long as you add the firs DVD ISO file and
the ISO file for the second DVD is located in the same directory.

Basically you only need the following repository description in your
config.xml file:

 <repository alias="SLES_11_SP1_DVD" type="yast2" priority="98">

      <source
path="iso:///platform/kiwi-repos/SLES/SLES-11-SP1-DVD-x86_64-GM-DVD1.iso" />

 </repository>

All the metadata needed is in this ISO and zypper will find the packages
from the second ISO file automatically as long as the second ISO is in
the same directory, which is is based on your earlier message.

Please try this and report back.
