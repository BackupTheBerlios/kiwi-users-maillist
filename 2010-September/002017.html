Hi,

> Hi,
> 
> I have package repositories that contain multiple versions of the same
> package and I want to reference these repos for KIWI image building. Is
> there anyway to specify a specific version of a package to be used in a
> KIWI image? I did not see anything in the KIWI schema that would do
> this.

yes you only have to full qualify the package like.
Let's say you have "vim" in your package list

  <package name="vim"/>

But you want the vim version 7.2-7.4.1:

  <package name="vim-7.2-7.4.1"/>

you have to add the name like it is displayed if you call
'rpm -qp package-name'. The package manager will use exactly
this version of the package and will fail if it does not
exist
