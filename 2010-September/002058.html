> > I am using kiwi to create VMWare format VMDK and VMX. I can play it using
> VMPlayer, however, I could not run the VM in ESX without converting it using
> VCenter Converter. Is there a way to make ESX/i compatible VMDK using kiwi
> or is there a command line tool that could convert it?
> 
> iirc ESX requires the image to be in the ovf format. If you have the ovftool for
> Linux installed:
> 
>   http://www.vmware.com/appliances/getting-started/learn/ovf.html
> 
> you can change your xml from
> 
>    format="vmdk"
> 
> into
> 
>    format="ovf"
> 
> kiwi will then call ovftool to create an image in ovf format which should run
> on ESX too
> 
> the ovftool from VMware is a binary only tool which we are not allowed to
> distribute so you need to manage the installation yourself.
