Changes Report kiwi v4.41 2010-06-06 - 2010-06-11
--------------------------------------------------
most important changes for this weeks release
not many interesting things due to hack week in NUE though :-)
Here we go:

GIT:  added oem-unattended element which allows unattended
      installation if possible

   <oemconfig>
      <oem-unattended>true</oem-unattended>
   </oemconfig>

   This will prevent any questins when installing an image via
   the oem install iso or stick created by kiwi. Be careful
   with this option because it will also prevent the question
   whether you are sure to wipe your disk or not :-) 

GIT:  added oem-dumphalt element which reboots the system after
      the image was dumped (installed) on the target device. A message
      appears and the system waits for a key event before reboot

   <oemconfig>
      <oem-dumphalt>true</oem-dumphalt>
   </oemconfig>

   causes a reboot after the installation via an oem installable
   iso or stick.
