> Hi all,
> 
> I am trying to create a repo, either yum or inst-source, from a kiwi desc of a livecd (the goal is to be able to use the dist-upgrade mechanism of a previous installation). Any hints where to look of whether is that possible with kiwi?
> 

I think you want a solved list of packages required for a specific image
This information can be obtained as follows:

   kiwi --info /path/to/image/description --select packages

if you have collected all packages the creation of a repo should
be done by using createrepo. The resulting repo can be used with
type="rpm-md" inside kiwi again
