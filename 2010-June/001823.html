>   I'd like to make a usb stick from a livecd created by kiwi. Is that
>   possible? I guess I should be using syslinux but I am not sure about
>   the parameters and config options.

I suggest to create a hybrid iso this will then use isolinux and
will work as iso and as disk


  <type image="iso" flags="clic" hybrid="true" hybridpersistent="true" boot="isoboot/suse-11.2"/>

hybrid support works with 11.2 (syslinux from 11.2). The result is
a .iso file which you can deploy in several ways:

1) on a stick with
   
   dd if=file.iso of=/dev/stickdevice

2) as a live CD

   k3b file.iso

The hybridpersistent attribute works only for usb sticks where it
will create a write partition on the stick if possible and store
data persistently there instead of using RAM space
