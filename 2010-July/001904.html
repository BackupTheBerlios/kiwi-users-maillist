Changes Report kiwi v4.48 2010-07-11 - 2010-07-16
--------------------------------------------------
most important changes for this weeks release...

GIT:  - fixed recovery cleanup code
      - fixed umountSystem call to check proc/mounts and umount
        a list of possible system mount paths in the right order
      - netboot: fixed creation of read-write partition
      - netboot: fixed boot partition setup, IFS restore required
      - netboot: imageDiskDevice was not set but used

 fixed pxe images when clicfs as filesystem is used

GIT:  - fixed race condition resulting at times in message of not
        finding the MBR and other times not finding the CD drive.

 finally Robert and Alex found this problem. Problem was the
 missing wait period for udev to create the device nodes

many small bugfixes in this weeks release which I didn't
mention here. For details about them watch the git commit log

* I'm now working to add btrfs support for 11.3 images
* We are working to have 11.3 support in Studio soon

Have Fun :-)
