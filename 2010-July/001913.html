On Tue, 2010-07-20 at 15:14 -0300, Chris Bowlby wrote:

> Also, where can I enable the kiwidebug=1 variable so that we can gain
> shell access, things are occurring too quickly after the PXE image
> booting phase to be able to catch a spot to enter it? I suspect it
> would be in the pxelinux.cfg/default file, but just want to confirm
> first.

It goes on the command line to the kernel. So it is
in /tftpboot/.../pxelinux.cfg/default. Mine is:
        
 LABEL KIWI-Boot
   kernel rsoft/linux-w
   append initrd=rsoft/initrd-w vga=791 ramdisk_size=512000 ramdisk_blocksize=4096 kiwidebug=1
   IPAPPEND 2
