> unfortunately for the createtion of a openSUSE 11.3 Live USB stick
> in / doc/packages/kiwi/exampels/suse-11.3 exists no example
> directory for creating a USB stick. With the use and adaptation of
> the Directory of 11.2, it is me so far failed to create a working
> Stick. What is important in the creation and adaptation of the
> config files for 11.3?

basically you should be able to build the 11.2 description for
11.3 too. Maybe you need to change this or that package but it
shouldn't be too much work

If you want to start with a JeOS as template for the stick
you can do the following to build it:

1) make sure kiwi-templates is installed
2) call kiwi:

   kiwi --build suse-11.3-JeOS -d /tmp/myimage

   * optionally set a local repo to speed up the build with --set-repo
   * optionally add hybridpersistent="true" to
     /usr/share/kiwi/image/suse-11.3-JeOS/config.xml to activate
     persistent writing on the stick

3) after the build is done deploy the generated iso image
   on the stick:

   dd if=file.iso of=/dev/your-stick

This works because the iso is a hybrid iso which can be either
burned on CD or dumped on stick. It's the currently preferred method
for USB sticks.
