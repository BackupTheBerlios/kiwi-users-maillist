I moved from Thinstation to KIWI a while back. I also have need to
classing equipment into various groups. I have found that I can use the
domain-name variable. I have attached my dhcpd.conf file. I made each
class of hardware into a group. Then I just add system to the group I
want them to be in. The dhcpd.conf file is the only place I need to
specify anything about any system.

In KIWI, I start the various services via a single rc script. It sees
the domain-name as DNSDOMAIN. This is part of my rc script:

      case "${DNSDOMAIN}" in

              mca)    echo -n "for MCA "
                      start_mca
                      ;;
              jboc)   echo -n "for just a bunch of computers "
                      start_jboc
                      ;;
              dpavue) echo -n "for digital PAVUE "    
                      start_dpavue
                      ;;
              rangere55) echo -n "for SICK Ranger E55 "  
                         start_rangere55
                        ;;
              test)   echo -n "for testing "  
                      start_test
                      ;;
              *)      echo -n "however, ${DNSDOMAIN} is not recognized "
                      ;;
      esac
