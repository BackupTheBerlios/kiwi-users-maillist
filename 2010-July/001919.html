Changes Report kiwi v4.50 2010-07-18 - 2010-07-23
--------------------------------------------------
most important changes for this weeks kiwi release...

GIT:  fixed URL quoting, we have to distinguish the quoting
      for local path URL's and network protocol URL's

 if you use a remote URL (http,https,etc..) and the URL contains
 special characters like "$" they will be escapted which is wrong
 for remote URL's but valid for local path URLs


GIT:  fixed oem-partition-install for LVM images

 if the <oem-partition-install> feature is used in combination
 with lvm="true" you need this version in order to make the
 deployment process work

GIT:  fixed locale validation and documentation as well as
      some template examples provided by kiwi. The locale value
      in kiwi is always a UTF-8 locale therefore it's not
      allowed to add the encoding type to the locale string.
      We also support the xx_XX locale style only which is
      now all checked by the schema (bnc #620789)

 *** commit message explains it imho ***

GIT:  added 11.3 as migration target

 just give it a try: 'kiwi --migrate mySystem' :-)
 the module is at least good to inspect your system's integrity
 
GIT   fixed oem recovery for LVM based images. It's required
      to backup the LVM metadata in order to restore it

 if the image is a LVM based image and <oem-recovery> is used
 it's required to store the LVM metadata in order to allow
 restore from really destroyed partitions


We are switching to a newer kiwi version for Studio in the
next weeks release and therefore 11.3 support will work
soon. Wish you all a great weekend

Remember to have fun :-)
