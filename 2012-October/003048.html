On 10/04/2012 03:01 PM, Badjatya, Nikunj wrote:
> Howdy All,
>
> I need to create a SLES 11 SP2 64bit VMware (ovf) image. I have few
> custom software as .rpm as well as .tar.
>
> Rpm can be added under <packages> in config.xml file. But for installing
> .tar, our traditional way to install .tar is untar it first, run the
> scripts which came after untarring.
> Can this .tar install be automated using Kiwi?

Use the <archive> element, see 
http://doc.opensuse.org/projects/kiwi/schema-doc/ for details.

The tar archive must be located at the same level as the config.xml 
file. You can then execute any scripts that may be part of the tar 
archive from the config.sh file. Note any scripts executed must be fully 
automated, i.e. it is not possible to have any interaction. I suppose 
you can use expect if you neet to interact with any of the install scripts.
