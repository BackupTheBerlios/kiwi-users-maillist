Hi All,

==========>

this is an important change for all of you so please read
the entire mail. Thanks 

==========>
!!! The following applies effectively by today !!!
==========>

because berlios will close at the end of this year I have moved
the kiwi source and branches over to github. The new project
page is here:

   https://github.com/openSUSE/kiwi

----------------------------------------
Mailing list
----------------------------------------
The mailing lists on berlios will close down as well.
Therefore I set up an open group named kiwi on googlegroups
which everybody can join. I'd like to ask all of you who
have interest in kiwi to join the new group. I will continue
to read and answer questions on berlios as long as it
works though. Mails from the following group take the
prefix '[openSUSE/kiwi]' so that you can add a filter easily

   http://groups.google.com/group/kiwi-images

----------------------------------------
Move the berlios checkout to github
----------------------------------------
if you have a berlios checkout you can easily move the origin
url by editing your project .git/config file and change it
to

  [remote "origin"]
    url = ssh://git@github.com/openSUSE/kiwi.git
    fetch = +refs/heads/*:refs/remotes/origin/*

----------------------------------------
Checkout the code from scratch
----------------------------------------
In order to checkout the code the command is as follows:

  git clone git at github.com:openSUSE/kiwi.git

----------------------------------------
Collaboration
----------------------------------------
For collaboration there are now two ways:

1) use github's pull request mechanism. Read more about
   this here: http://help.github.com/send-pull-requests/

2) or simply proceed writing git patch mails to the
   following address: kiwi-images at googlegroups.com

I'm sorry for any inconvenience and I'm looking forward
to see all of you in the new kiwi google group :)

Best Regards,
Marcus
