Hi,

> >I'm hoping to be able to create a multi-partition system, something like:
> >
> >/, size 50 G
> >/var, size 200G
> >/tmp, size 100G
> >/log, rest of disk
> >
> >Is that something I should attempt in .../source/root/build_custom, or
> >in .../source/root/etc/init.d/suse_studio_firstboot ?
> >
> >(Obviously I started with a SUSEstudio app and have been hacking on it
> >ever since.)
> 
> For multiple partitions you will need to use LVM. KIWI and Studio do
> not support the partitioning of the root partition, but you can
> create volumes to your liking. Take a look at the <systemdisk>
> element in the schema doc.

yep in XML speak:

   <type ...
      <systemdisk>
        <volume name="var" size="200G"/>
		<volume name="tmp" size="100G"/>
        <volume name="log" freespace="all"/>
      </systemdisk>
   </type>
