On 05/08/2012 06:19 PM, Eric Richards wrote:
> Hi, I have another newbie question for you: Is there a reference manual
> for the config.xml schema?

Yes, /usr/share/doc/packages/kiwi/schema/kiwi.html part of the kiwi-doc 
package.

>
> I'm hoping to be able to create a multi-partition system, something like:
>
> /, size 50 G
> /var, size 200G
> /tmp, size 100G
> /log, rest of disk
>
> Is that something I should attempt in .../source/root/build_custom, or
> in .../source/root/etc/init.d/suse_studio_firstboot ?
>
> (Obviously I started with a SUSEstudio app and have been hacking on it
> ever since.)

For multiple partitions you will need to use LVM. KIWI and Studio do not 
support the partitioning of the root partition, but you can create 
volumes to your liking. Take a look at the <systemdisk> element in the 
schema doc.
