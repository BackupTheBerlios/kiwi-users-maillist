ah I see ok. In kiwi there is a check if the created partition and
filesystem can hold the files which are about to added there.
if not (due to filesystem meta data, journal size and partition alignment)
kiwi try it again with a +100MB disk size until everything fits
into it.

Maybe we should decrease the stepsize here to let's say 10MB.
So change the line to:

   $this -> __updateDiskSize (10);

and see if that brings you any further
