Hi,

> If we want to lay down a partition not created by kiwi, is there a way
> to do that?
> 
> The use case is to lay down a dos partition with system diag tools. Any
> ideas on how to accomplish this?

unfortunately this is not possible. The oem image is a virtual disk
this means one file is representing the entire system including
boot record, partition table, and data. So as soon as this image is
installed all data in the disk is lost.

Possible is to create a dos partition on first boot and add the
tools. It also would be possible to setup the oem disk with an empty
dos partition which allows you to store data before the first boot
of the image happens

In any of these cases an implementation to the current oem code
is required
