you are right we don't have that in kiwi. I guess it also would be a very
distro specific mapping which also will change from each distro version
to the next. I'm not sure but does yast not have a module or functionality
to install language sets ? I think it would be a good idea to use that
functionality instead of implementing and maintainng it twice

* It's easy to integrate yast firstboot and/or autoyast setup into kiwi
  so if there is a module which we can pass a language to and it installs
  everything required we are fine I guess

* Another solution is to create a language profile section in your
  config.xml

     <packages type="image" profiles="de_DE">
         <package name="..."/>
         <package name="..."/>
     </packages>

  You can run kiwi with the option --add-profile de_DE and it will install
  the mentioned packages of this section. That would work today already
  You also can have multiple of these profiles packages sections and can
  select also more than one to become included
