On Sunday 04 January 2009 19:56:50 znmeb at cesmail.net wrote:
> 1. I added the repositories, but the packages there just seem to be a
> README file. Are there other packages somewhere else that actually
> build the USB images?

The package which builds the ISO is kiwi-image-livecd-kde3 (LiveCD) and kiwi-
image-usb-kde3 (LiveUSB). Note that you won't be able to build it using build 
service because you don't have root access, like I don't have either so in 
order to build you will have to build locally:

osc co home:cgoncalves:LiveCD kiwi-image-livecd-kde3
osc build --userootforbuild PLATFORM ARCH kiwi-image-livecd-kde3.spec

> 2. I am downloading the KDE 3 USB image from the site now. How do I
> install it? Do I unzip the "gz" and just copy the ".raw" file directly
> to "/dev/sdb" with "dd"??

gzip -cd <download>.raw.gz | dd of=/dev/<device_name> bs=4096
