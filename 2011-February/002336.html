ok, now I understand. You see this only for the creation of the boot
image (initrd). oemboot/suse-11.2. I remember we updated kiwi-tools
and because this happened after 11.2 was done we use the sle11 version
of the kiwi-tools for the boot image. That works because it contains
only simple C helper programs (no deep library dependencies)

I though you added that repo for your system image which wouldn't make
much sense imho. The kiwi-tools package is only relevant for the
boot code. It contains small helper programs important when the system
boots up but you don't need them in the system itself
