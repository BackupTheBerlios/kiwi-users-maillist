Marcin,

squashfs and aufs are only available for kernels in 10.2 and the 
upcoming 10.3 release. You can still create ISO images, you just cannot 
compress the file system.

Do not use the "flags" attribute in the "type" element and Kiwi will 
create a non compressed file system. See section 8.3 in 
/usr/share/doc/packages/kiwi/kiwi.pdf

Robert
