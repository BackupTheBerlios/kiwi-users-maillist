> Here are the OEM options used.  I think I am more confused than before what 
> <size/> means.
>
> <oem-boot-title>A string</oem-boot-title>
> <oem-reboot>yes</oem-reboot>
> <oem-swap>yes</oem-swap>
> <!-- 128 GB swap partition -->
> <oem-swapsize>131072</oem-swapsize>

ok by default the repartition code will create a 10GB root
filesystem and will use the rest - swap for the home partition where
people need the space. If you want another size for the root
partition use

   <oem-systemsize>...</oem-systemsize>

if you don't want to take care for the root size at all just
set:

   <oem-home>no</oem-home>

this will prevent an extra home partition and will simply use
all the space - swap
