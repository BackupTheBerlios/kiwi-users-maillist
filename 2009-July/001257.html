Hi,

> However, I do not understand/know how to use my own theme, i.e. 
> usr/share/gfxboot/themes/myBrand instead of 
> usr/share/gfxboot/themes/openSUSE

- you need to provide all data according to an existing theme
  in your own myBrand directory first

- then set:

  <boot-theme>myBrand</boot-theme>

  in config.xml

> Second when creating an ISO image I still get the openSUSE welcome 
> screen, how can this be customized, avoided?

you can't avoid the welcome message but brand it:

/usr/share/gfxboot/themes/SuSE/install/welcome.jpg

> Third, when booting no image is shown, simply a black background with 
> the kernel message scrolling by. Is the an ISO image oddity or would 
> this be a problem with an installed image as well?

You are talking about the splash screen. If you can't see
it your image doesn't include something like

<package name="bootsplash-branding-openSUSE" bootinclude="true" bootdelete="true"/>

in config.xml. To brand the splash screen the data in

/etc/bootsplash/themes/openSUSE/images/

needs to be changed
