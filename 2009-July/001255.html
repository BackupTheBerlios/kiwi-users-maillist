> Yes you should be able to do this. One approach could be to use Kiwi 
> to create 2 raw images, these are disk images one can dump onto a hard 
> drive using dd. You can then "embed" both .raw files into a LiveDVD 
> image created by Kiwi. From this Live system you can easily display a 
> GUI for the user to select whether to install a desktop or server. 
> Behind the scene you simply run "dd". You will have to do some device 
> management if there are multiple drives in a machine. The tricky part 
> will be in the details w.r.t. the hardware configuration. If the 
> hardware configuration is known and always the same this is all 
> relatively simple.
