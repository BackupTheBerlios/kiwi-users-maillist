Hi,

> But what I want is 10 GB root, 128 GB swap and the rest of the 
> drive unused to have space to create a LVM volume group.

This should work like the following:

<oem-systemsize>10240<oem-systemsize>
<oem-home>no</oem-home>
<oem-swapsize>131072</oem-swapsize>

- create a 10GB root partition including home
- create a 128GB swap partition
- prevent creation of home partition which causes
  some spare space left on the disk
