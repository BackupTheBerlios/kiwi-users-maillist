Hmm, ok... well to run yast I touch the file /var/lib/YaST2/runme_at_boot
and everything else is yast magic. I thought the script would setup the
network if needed but maybe it doesn't. There is another mechanism called
yast firstboot. I don't know how it works but the yast people said it's
the preferred.

I suggest to contact Duncan Mac-Vicar Prett <dmacvicar at novell.com>
and ask for help on this. IIRC we already make use this system in another
project

of course you can fake network startup in some early boot script but
I would say that's a hack :)
