Hi,

> I want to add software to the live CD. Is this possible with KIWI or
> would I have to do it from scratch?

Do you mean you want to add software to a running liveCD or do you
want to add software to the live CD image description ?

I guess you want to install software to a running liveCD

* this is possible if your live system is a "unified" live system which
  means it is based on aufs. In this case the complete root / is writable
  But the software will be installed into RAM and is not persistent.
  
* in any other case the live system will create links to specific
  read-only directories. These are /bin /boot /lib /opt /sbin /usr
  If your software gets installed to one of these directories it will
  fail because they are read-only on the CD or DVD
