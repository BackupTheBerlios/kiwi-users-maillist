Hi folks,

I ran into trouble today trying to resolve "opensuse://<project> paths.
According to Adrian the path must be set differently.

I prepared a patch that fixes this; it's just one variable in kiwi.pl pointing 
to the wrong server. It is accessed in modules/KIWIURL.pm in the method 
openSUSEpath.

Another small thing in the patch is the problem with forking the Perl 
debugger, which fails to get a TTY when the KIWILog forks.
The small hack in the beginning ($DB::...) fixes that, so for those who use 
the debugger (I know, debugging is for losers *g*)

Marcus, I can create a different patch without the -d if you wish.

The other patch fixes my KIWIXML::splitPathHTTP method which fails if the base 
URL doesn't return a dir listing but something different.
