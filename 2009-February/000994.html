Hi All,

the current kiwi uses grub to boot USB sticks. Many people
had problems booting with grub via ext2. As an alternative
I have implemented syslinux via fat which should work on
nearly all BIOS implementations and USB sticks. If you see
problems with your USB stick you can try to use:

   <type ... bootloader="syslinux">usb</type>

The feature also works for vmx and oem types but when
booting this image types for the first time the linuxrc
bootloader setup will re-introduce grub again. I need to
enhance that in a next version

Hope you find it useful
