Hello Community

I am happy to announce the release of KIWI-LTSP 0.7.5 based on LTSP
5.1.65 using KIWI v3.34. Major improvements since the last
announcement are as follows:

* Support for building full featured "Fat" client images
(kiwi-ltsp-setup -fat), images based on KDE4, KDE3 and Gnome from
easy-kiwi-desc can be used as AOE exported client images[1]. Requires
LDAP client configured on LTSP server.
* Full IceWM session now available as LTSP-Localapps in
kiwi-ltsp-prebuilt image[2].
* Simple load-balancing cluster using get_host script available by default.[3]
* AOE support with easy configuration for multiple images apart from
NFS and NBD for booting LTSP or Fat image. [4]

Improvements this time around are thanks to Shrenik "Shrek" Bhura, J
"Slimer" Nelson, Marcus "schaefi" Schäfer, Gideon "Gadi" Romm and of
course everyone else of LTSP and KIWI team :)

I would also like to share the news that Jan "The_Code" Weber has
started re-write of Easy-LTSP[5] (easy-ltsp-ng) in Python so hopefully
more distributions will get benefit of that cool tool.

Suggestions, feedbacks welcome, bugs and enhancement go here
http://devzilla.novell.com/kiwi-ltsp/

Remember to have fun...

Jigish
