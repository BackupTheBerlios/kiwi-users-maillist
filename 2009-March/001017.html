> I originally did have diskless. But I did not get the kernel-pae. It was
> suggested that to get the -pae kernel, I should use smp instead. I have

I now understand all the details of the problem. You use the
diskless profile which always uses the default kernel additionally
you added (per advise) the smp profile which also adds the pae
kernel. So you ended up with two kernels

I'm going to fix this:

a) changing the profile name smp to pae for >= 11.1
b) adding kernel profiles for diskless named
   diskless-kdefault and diskless-kpae

For you this means you should change the bootprofile
attribute to the following:

   bootprofile="diskless,diskless-pae"

which means use the diskless drivers,packages and the pae
kernel which is I think what you want

I will introduce the change with the next kiwi version
v3.32

===========================
!!! For the LTSP people !!!
===========================
according to that change you need to make sure to
adapt your bootprofile setup. It's required to select
a kernel profile now. The setup:

   bootprofile="diskless"

needs to be changed to

   bootprofile="diskless,diskless-kdefault"

to have the same behavior as before

Thanks and sorry for any inconvenience
