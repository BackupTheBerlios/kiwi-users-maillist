Hi,

> > Thanks. Do you know if I Sax2 will ignore the "-m 0=nvidia" switch on a
> > machines that do not have a nvidia video card? If Sax2 uses the switch
> > regardless, I think I will need to customise the startx script or add a script
> > to boot.local that detects if nvidia is installed (lspci perhaps)?
> 
> No it will not ignore it, sax2 -a might use nvidia if the hardware is
> found and driver installed.

correct. if you set the -m option you specifically set a module
and this is done in any case. You can provide an update Identity
map for the card which is not detected correctly

    /usr/share/sax/sysp/maps/update/

you should have some maps there already, if not take a look at
the main map:

    /usr/share/sax/sysp/maps/Identity.map
