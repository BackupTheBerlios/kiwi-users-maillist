> get some trouble. Any advise how to solve this problem or what causes the

Those parts of kiwi which are distribution dependent are prefixed with
a "suse". This includes for example some functions in KIWIConfig.sh
as well as the entire boot code which is different in every distro:

   suse-linuxrc
   suse-preinit

The boot code includes the KIWILinuxRC.sh as common set of functions
which should work on every distro.
