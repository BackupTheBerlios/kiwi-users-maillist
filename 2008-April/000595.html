Config.xml just includes the default "linux" and "root" user accounts. 
/var/log/messages logs an account named "festival", among others. Most 
are described as daemon accounts. An example:

04/23/2008 04:24:18 PM	linux	shadow[19619]	running GROUPADD_CMD command 
- script=/usr/sbin/groupadd.local, account=festival, uid=103, gid=0, 
home=, by=0
04/23/2008 04:24:18 PM	linux	useradd[19621]	new account added - 
account=festival, uid=101, gid=103, home=/usr/share/festival/, 
shell=/bin/false, by=0
