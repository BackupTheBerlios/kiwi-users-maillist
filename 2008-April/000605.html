The only packages that were needed was smart package mananger, and
squashfs. Of course this was after I installed dhcp-server, and tftp
server from the SLES cd. 

It took me less than an hour to get up and going, I have this problem
when it gets to the KIWI imaging part:

Checking Server name: 10.0.0.254
Server : 10.0.0.254 not found
Using: 0.0.0.0 from dhcpdcd-info

At this point it wouldn't boot any further. Mind you this is straight
out of the box, with some messing around with the dhcpd.conf I was able
to get it to say the following:

Checking Server name: 10.0.0.254
Server : 10.0.0.254 not found
Using: 10.17.32.6 from dhcpdcd-info

I double checked /etc/sysconfig/kiwi-ltsp to make sure that all IP
addresses were correct, is there another config file somewhere that I
can check?

And from there it went on with the boot process. Seems to be very good
so far.
