Hi,

> is there a way to use kiwi to migrate a runnig system into a live-cd system.
> I ve a working nagios-monitoring-system and  I m looking for an easy to
> use way to convert this runnig system.
> Something like starting a script and some minutes later a working iso is
>  written into a directory.

there is experimental support for migrating a running system into
an image description from which you can create each of the supported
image types.

	kiwi --migrate mySystem --destdir /tmp/bob [ --exclude <dir> ]
