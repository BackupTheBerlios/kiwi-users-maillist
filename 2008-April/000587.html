Marcus Schäfer wrote:
> Hi,
> 
>> How do you point Kiwi to the mounted ISO? The only reference I can find 
>> in the documentation or the config file is in the repos. If I direct 
>> Kiwi to the repo directory (/mnt/iso/suse) it fails to find the media 
>> file. If I direct it one level up (/mnt/iso) it fails to find the packages.
> 
> could you send the complete repository definition ? the repo information
> is used to feed the packagemanager so along with the path the repo type
> must also fit. If you use a suse CD as repo you need to use the repo
> type "yast2" and point the path to the directory which contains the
> "suse" subdirectory
> 
> Regards,
> Marcus

The iso for the openSUSE CD is mounted on /mnt/iso. The repos are

   <repository type="yast2">
     <source path="/mnt/iso/suse"/>
   </repository>
   <repository type="rpm-md">
     <source path="http://download.opensuse.org/update/10.3/"/>
   </repository>
   <repository type="rpm-dir">
     <source path="/usr/share/kiwi/image/kwliveCD-suse-10.3/repo" />
   </repository>

In this scheme, Kiwi looks for the media file in /mnt/iso/suse/media.1 
and fails in the initialization.
