Well I spent some time to make the packages build and I cleaned the
code to be ready for further implementations which could be arch
specific. One of these missing implementations is support for other
bootloaders than grub. On ppc another boot loader is used and along
with the bootloader background graphics another system than gfxboot
is used. It simply doesn't exist on ppc. I'm sorry but I had no
time to adapt the image descriptions for ppc and I also did not find
time to make the boot setup work for ppc. There is also s390 which
requested basic root image tree support first.

positions so I would be glad to see people join :-)
