Hi,

> Running this:
> 
>  kiwi --bootstick
> /mnt/sdb5/usb-test/initrd-usbboot-suse-11.0.i686-2.1.1.splash.gz \
>  --bootstick-system /mnt/sdb5/usb-test/usb-test.i686-1.0.0
> --bootstick-device /dev/sdc
> 
> Kiwi says : "Couldn't find any removable USB storage devices"
> 
> Here is what fdisk shows:
> 
> Disk /dev/sdc: 8178 MB, 8178892800 bytes
> 252 heads, 62 sectors/track, 1022 cylinders
> Units = cylinders of 15624 * 512 = 7999488 bytes
> Disk identifier: 0x00000000

The usbboot boot image type is for USB stick systems only. The initrd
code searches for disks which has the removable=1 flag set which doesn't
apply to your disk (flash disk). I think usbboot boot image is the
wrong choice for your setup. This looks more like an oem to me so
oemboot might be better.
