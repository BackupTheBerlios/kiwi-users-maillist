Hi,

> What's the easiest way to install a custom kernel for a live cd/dvd?

so far we have kernel profiles in isoboot for:

 std => kernel-default
 pae => kernel-pae
 rt  => kernel-rt
 nex => kernel-xen

you can select them with the bootkernel="..." in your
system image description.

if none of the above matches your kernel I suggest the
following way to integrate your own kernel (I guess you want
to create an image for suse-11.1):

1) clone the appropriate isoboot boot image description

   cd /usr/share/kiwi/image/isoboot
   cp -a suse-11.1 suse-11.1-mike
   rm -f suse-11.1-mike/.checksum.md5
   
2) edit the cloned description

   vi suse-11.1-mike/config.xml

   ===> change for example the std profiled kernel
        and replace it with the name of your own package

3) reference suse-11.1-mike instead of suse-11.1 in your
   system image description:

   <type ... boot="suse-11.1-mike">iso</type>

4) make sure your system image description also includes
   your version of the kernel and therefore ensure that
   the system image have the same kernel than the boot
   image

5) don't open bug reports with custom kernels :-))
  
That's it
