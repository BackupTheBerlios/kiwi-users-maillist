> Thanks for your assistance everything now work correctly. just a quick
> question i read in Kiwi man page that i can create an usb disc for
> installing a system like i did with my oem version on dvd using this command
> kiwi --installstick Initrd --installstick-system VMXSystemImage
> that is correct or i misunderstood ??

this is not quite correct. If you want the oem type to create
an install USB stick instead of an install CD/DVD you only need
to change the format attribute from:
 
   format="iso" 

to

   format="usb"

when calling kiwi create with --type oem the result will
create an additional image with the extension:

   .install.raw

You can dump (dd) this image on the stick and it will
behave the same way as the install iso image

The usb image type is meant for live sticks where you
work from the stick. If the man page doesn't make that
clear we really should fix the documentation. Feel free
to send any suggestions on how to make the docs better

There is also the kiwi-doc package which includes the
full doc (kiwi.pdf) where I try to exlpain each image
type more detailed
