> How can I select the packages from SLES10SP2 to Live CD? thank.

I guess the system you run kiwi on is also a SLES10 system.
The easiest way to do this is the following:

1) install the smart package manager from here:
   http://download.opensuse.org/repositories/smart/SLE_10/

2) install the kiwi packages from here:
   http://download.opensuse.org/repositories/Virtualization:/Appliances/SLE_10/

3) make sure you have installed the kiwi-templates packages too

4) run kiwi:

   kiwi -b suse-SLE10-JeOS -d /tmp/myimage --type iso \
        --set-repo /path/to/sles10/repo

5) find the result in /tmp/myimage

Notes:
------
* the default zypper version from sles10 is too old to work properly
  with kiwi, that's the reason why I suggest to use the smart
  packagemanager

* if kiwi uses smart kiwi resolves patterns itself by using the
  satsolver library. The suse-SLE10-JeOS template makes use of
  the "base" pattern

If you still can't build the iso it would be great if I can
login to your machine and have a closer look
