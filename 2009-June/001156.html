kiwi version including the kiwi-templates package are you able to
simply build one of the templates by calling:

   mkdir /tmp/testbuild
   kiwi --build suse-11.1-JeOS -d /tmp/testbuild \
        [ --set-repo /path/to/11.1/repo ]

* If you don't set the repo this call will connect to the
  internet.

* If this works it must be something inside your image
  description. In that case it would be good if you post
  your image description

* And last but not least I strongly recommend to _not_
  change anything in the kiwi provided boot image descriptions
  below /usr/share/kiwi/image/vmxboot/. If you do changes
  here I can't guarantee that the result still boots as it
