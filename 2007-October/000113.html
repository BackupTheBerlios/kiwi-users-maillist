for xen you need to add an additional package section which describes
what packages must be added to your system image to allow this image
to run in a Xen virtual machine. In this section you also describe the
guest station parameters like memory or disk device. It could look like
the following example:

    <packages type="xen" memory="512" disk="/dev/sda">
        <package name="kernel-xen"/>
        <package name="xen"/>
    </packages>
