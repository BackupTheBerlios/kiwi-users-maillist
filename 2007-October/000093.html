> How can I change the image size after the physical extend has been
> created? I tried already in the .profile but is seems to be ignored. Is
> there any other place where the size is set?

All meta data is in:

	path/to/root/tree/image/

if size is set at all you can find it in:

	path/to/root/tree/image/config.xml
