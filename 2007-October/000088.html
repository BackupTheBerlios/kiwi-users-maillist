Hi Wolfgang,

> how is it with the runlevel setting on a network client? Can I just copy
> the runlevel setting from an normal installed system or are there some
> services which better should not be started on a kiwi client. 

I think the /etc/inittab where the default runlevel is set can
be simply copied into the system

> I'm thinking about the boot stuff (boot.localnet
> boot.loadmodules....etc) but also all other services. Is there anything
> I have to watch out?

for suse kiwi provides a function called suseActivateDefaultServices()
this will run the post scripts of your installed packages and activate
the default services. If you need other services too just activate them
with the function suseInsertService()
