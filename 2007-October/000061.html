Hi,

> I would like to have some clarification on PXE deployment via KIWI:
> Is it possible to deploy a 1GB image via KIWI on a diskfull system that have only 512MB of RAM? 
> 
> As I understand the way KIWI works, the image is first downloaded into the memory and then installed onto the disk thus making it not possible, but maybe I'm mistaken.

No, the download happens directly to the specified device of your
config.<MAC> file. If you have setup for example:

	DISK=/dev/sda
	PART=100;S;x,x;83;x
	IMAGE=/dev/sda2;your-image;your-image-version;IP.of.tftp.server;4096

the download takes place from the network into /dev/sda2. It would be
a very bad design if we download everything into ram first :-)
