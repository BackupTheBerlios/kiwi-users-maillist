LVM example:

1) install kiwi-templates
2) apply the LVM configuration patch I have attached to this
   mail via

   patch -p0 < template.dif

   It will activate lvm and some lvm volumes setup as example
   to the suse-11.2-JeOS template config.xml file

3) build the image in one shot

   mkdir /tmp/myimage
   kiwi --build suse-11.2-JeOS --set-repo /path/to/11.2/repo \
        -d /tmp/myimage --type vmx

I recommend to have a local copy of a 11.2 DVD in order to speed
up the build. If the image is done you can test it inside VMware or
KVM. I prefer kvm

If that works we can go a step further and check what we can
do with the luks encryption :)

Hope that helps
