> >
> Ignore that error, it is a bug in the script, check "ps ax | grep
> vblade" to see if the service is running, if not then start it
> manually:
>
>  vbladed 0 1 eth0 /path/to/imagename.img
>
> Change eth0 to the device you are using and 0 1 to whatever numbers
> used in /srv/tftpboot/KIWI/config.MAC AOEROOT=/dev/etherd/e0.1
>
> BTW, this method is deprecated, LTSP upstream now has fatclient
> support using LDM, instructions for setting up image for that will be
> posted soon, I have copied this mail to the students working on that
> project to do the needful.
