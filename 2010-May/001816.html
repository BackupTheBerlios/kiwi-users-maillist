Changes Report kiwi v4.38 2010-05-23 - 2010-05-28
--------------------------------------------------
most important changes for last weeks release...

GIT: added support for extlinux based recovery (bnc #606651)
     make recovery work with separate boot partition
     use bind mount instead of link to boot partition

  kiwi now supports extlinux as bootloader. This change was the
  last in the chain of changes for this feature. In order to
  use extlinux instead of grub your type line has to include
  something like:

  <type .... bootloader="extlinux"/>

  in this case the following applies:

  * extlinux is used for system image
  * extlinux is used for recovery (oem only)
  * isolinux is used for install ISO (oem only)
  

GIT: changed ovftool options according to latest version

  The current version of ovftool has totally new set of options
  which I adapted in kiwi. Thanks for reporting this


GIT: fixed clicfs resize operation. The command is now based
     on a 512 byte sector size value instead of a 4k filesystem block
     size value which might be wrong depending on the size of the
     filesystem (bnc #604646)

  When we call resizefs we just pass a blocksize value and assume
  the blocksize is 4k. Depending on the size of the filesystem and
  also depending on the options given on create of the filesystem
  this assumption maybe wrong. So we changed this to pass a size
  indicator to the resizefs call as mentioned in the man page
