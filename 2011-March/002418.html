Hi,
 
I built an ISO with kiwi, now I would like to build PXE over read-only
NFS.
Everything works fine if my NFS server exports with "rw" option, my
system is alive and I can login but write to NFS.
 
My goal is to set "ro" option on my NFS export, combining with unionfs.
When I do ... "Can't write to ... read-only filesystem" juste after
mounting NFS.
 
My config.xml :
<preferences>
    <type image="pxe" filesystem="clicfs" boot="netboot/suse-SLES11"/>
    ...
</preferences>
...
<split>
  <temporary>
    <file name='/var'/>
    <file name='/var/*'/>
    <file name='/etc'/>
    <file name='/etc/*'/>
    <file name='/tmp'/>
    <file name='/tmp/*'/>
    <file name='/home'/>
    <file name='/home/*'/>
  </temporary>
</split>
...
 
My config.default :
NFSROOT=10.150.64.19;/home/nfs/sles/pxe
UNIONFS_CONFIG=/dev/ram1,nfs,aufs 
 
All documentation I red say to use UNIONFS_CONFIG with NFSROOT. What
have I missed ?
