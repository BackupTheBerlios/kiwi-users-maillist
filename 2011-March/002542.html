>> 
>>    /home/mhaase/appliance $ sudo kiwi -p source --force-new-root --set-repo /mnt/cdrom -r build
>>    /home/mhaase/appliance $ sudo kiwi -c build -d image --prebuiltbootimage bootimage -t iso

>> However the create step gives me the following error:
>> 
>> Couldn't locate configuration directory /usr/share/kiwi/image/initrd-oemboot-suse-11.4
> 
> yes because the boot image description can't be found according to your
> boot attr setup. This is not related to the use of a prebuilt boot image.
> kiwi checks if the boot image description exists. It does that to validate
> an eventual existing prebuilt version of that image description. So 
> you can't have a prebuilt version of the boot image without the appropriate
> image description for this prebuilt version
> 
> Your prebuilt setup looks good to me. I think it's just a matter of
> the wrong boot attr setup
