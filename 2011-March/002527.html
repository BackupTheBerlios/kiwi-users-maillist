You have a trailing "comment close" markup

<package name="bundle-lang-gnome-extras-en"/>-->

A couple of tips, install jing on your build machine, to get better 
error messages, don't insert blank lines as those cause nodes to be 
created during XML processing and might throw off the line numbers 
reported as error.

Run kiwi --check-config PATH_TO_config.xml

to verify the configuration file.
