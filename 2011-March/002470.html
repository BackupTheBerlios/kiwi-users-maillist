Hi,

> Is it possible to tell kiwi to only use a repository for a certain
> architecture?

beside profiles there is the %arch macro which you can put
in the url. Example:

  <repository type="yast2">
    <source path="http://download.suse.de/install/SLP/SLES-11-SP1-GM/%arch/DVD1/"/>
