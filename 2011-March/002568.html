On 03/29/2011 02:40 AM, Marcus Schäfer wrote:
> Hi,
>
>> Is there a way to change what looks like an encrypted password that is
>> being displayed in my config.xml file?
>>
>> The password that is now there is "linux" as this is what Suse Studio
>> uses as it's default.  Since my config.xml file originally was
>> exported from Studio and I didn't actually place this here I'm not
>> sure how to change/replace this password with something else.
>> Can this encrypted password in config.xml be changed to simply the
>> password that I want there or do I have to encrypt it first with
>> another application?
>
> simple passwords can be created with kiwi like this:
>
>     ./kiwi.pl --createpassword
>
> place the result hash in the config.xml
>
> you also can create a password hash using a better (more safe)
> program and place the result in the config.xml

Alternatively you can store the password in the config.xml file in plain 
text and then add the pwdformat attribute as shown in the following example:

<user pwd="linux" pwdformat="plain" home="/home/tux" name="tux"/>
