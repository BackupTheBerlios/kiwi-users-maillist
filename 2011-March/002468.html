Add the following repository to your system:

http://download.opensuse.org/repositories/Virtualization:/Appliances/

-> zypper ar -f 
http://download.opensuse.org/repositories/Virtualization:/Appliances/ 
kiwi_dev

Then look for the kiwi packages:

-> zypper se kiwi

Then install all the packages. You should end up with an installed list 
similar to this:

kiwi-config-openSUSE-11.4-6.6.3.x86_64
kiwi-debuginfo-4.43-1.6.x86_64
kiwi-desc-oemboot-4.80-214.1.noarch
kiwi-instsource-plugins-openSUSE-11-3-0.9-11.1.noarch
kiwi-debugsource-4.80-1.2.1.x86_64
kiwi-tools-4.81-215.1.x86_64
kiwi-pxeboot-prebuild-3.74-17.21.2.x86_64
kiwi-desc-vmxboot-4.80-214.1.noarch
kiwi-desc-netboot-4.80-214.1.noarch
kiwi-instsource-plugins-openSUSE-11-2-0.9-14.1.noarch
kiwi-templates-4.80-214.1.noarch
kiwi-desc-usbboot-4.80-214.1.noarch
kiwi-pxeboot-4.81-215.1.noarch
kiwi-desc-isoboot-4.80-214.1.noarch
kiwi-tools-imagewriter-3.74-17.21.2.x86_64
kiwi-instsource-plugins-openSUSE-11-1-2.0-23.1.noarch
kiwi-doc-4.81-215.1.noarch
