Hi,

> My questions are:
> 
> 1. Does kiwi just pass the pattern on to zypper and leave it to zypper
> to figure this out? I would imagine so.

yes this is correct if zypper is used as package manager

> 2. What is the best way to see what packages a specific pattern on a
> specific openSUSE release will cause to be installed? I guess this is
> not specifically a kiwi issue. But I think kiwi is a tool that works
> best when one uses patterns rather than specific package names. 


  kiwi --info /path/to/your/image/description --select packages

this will print the solved package list. If you substract the
packages you added directly you will see what the patterns
would put into the image

Other than that if the image is the same distro as your buildsystem
is you can run 'zypper patinfo name'
