> I have a very important requirement that I have to ask about.  The way
> that I'm currently using my kiwi build image is using a live ISO image. 

If you create an ISO image (hybrid) you do not have an option to create
the FAT filesystem at the beginning you are after.

What we have so far is that one create the FAT partition with a "usb"
image type (that is now going away, although I am not yet convinced that
that's a good idea) and one can create the FAT partition with the OEM
image type.
