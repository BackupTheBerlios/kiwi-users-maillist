> Is there a place where all the pattern names for a release can be found?

Yes,

On the media you will find

suse/setup/descr/patterns

This file contains the name of a file that contains the pattern
description, for oS 11.3 that would be dvd-11.3-22.1.x86_64.pat.gz

Inside, pattern names are indicated by the prefix =Pat:, thus

grep =Pat: dvd-11.3-22.1.x86_64.pat provides a list of all the available
patterns.
