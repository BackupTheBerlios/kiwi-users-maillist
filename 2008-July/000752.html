This image looks a little too minimal. No kernel, no bootloader, no rpm 
and the like.
Have you got the package kiwi-desc-misc installed? If so, please take a 
look at the image description 
/usr/share/kiwi//image/ctso-minimal-suse-10.3/config.xml


If you for instance include the pattern base-10.3, you'll get all 
packages the pattern requires. You can influence if the patterns 
Requirements (Prq), Recommendations (Prc) or even the Suggestions (Sug) 
shall be included.
Just add to the <packages> tag:
 <packages type="image" patternType="onlyRequired">
to avoid excess fat.

Then you could use the pattern like this:
<opensusePattern name="base-10.3"/>


This makes the package handling more convenient for you.
If you omit the "onlyRequired" then the required + recommended packages 
are installed IIRC.

I hope this helps solve your issues.
