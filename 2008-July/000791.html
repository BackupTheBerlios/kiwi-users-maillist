Hi,

this is just for people who are interested to play with the
Amazon EC2 web service:

	http://docs.amazonwebservices.com/AmazonEC2/gsg/2006-06-26/

It's pretty interesting project which allows you to use
web space and computing power at Amazon. They setup a system which
is based on Xen and allows you to remote control and run instances.
By design of this project it requires operating system images to
work. Amazon call these bundles

* kiwi is now able to create such bundles by using the tools provided
  with Amazon. We also plan to package and distribute their tools as
  soon as all license questions are clarified

* A "normal" user is only able to create system images. It is not
  possible to upload kernel and initrd (boot) images at Amazon if you
  are not a trusted party. Fortunately we managed to become such a
  trusted partner and I setup kernel and initrd images for

  openSUSE 11.0 and SLES10

  SLES10 requires a specific kernel which is not available at
  the moment but will be very soon. For testing I recommend to use
  openSUSE 11.0 because the kernel there works at Amazon without
  any further modifications

So if any of you are interested in ec2 and have already setup an
account for the service feel free to build your ami's with kiwi :-)
I have two running at the moment

version 2.60 will also come with a ec2 example image and a small
suse-ec2-guest.readme. I'm going to write a chapter in the cookbook
the next days
