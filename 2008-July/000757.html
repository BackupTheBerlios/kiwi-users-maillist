> 1) you can upgrade your previosly created image root tree (kiwi prepare)
>    with kiwi again. Let's say you have prepared your tree below /tmp/kiwi-test
>    the following will upgrade the tree using the update repo for 10.3
>
>       kiwi --upgrade /tmp/kiwi-test \
>            --add-repo http://download.opensuse.org/update/10.3 \
>            --add-repotype rpm-md
>
>    now you need to re-create the image from the updated tree as you
>    did before
>
>       kiwi --create /tmp/kiwi-test ...
>
>
