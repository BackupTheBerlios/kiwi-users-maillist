I don't have a blog, but this is what I've done.  Feel free to publish
it somewhere if you want.

We have an Intranet for our staff.  In the Staff Cafe we have 4 thin
clients set up specifically to access the Intranet.

I installed SuSE 11.0 and put Kiwi LTSP on it.  I changed the window
manager to icewm.  I chose icewm because it's light and easy to lock
down.  I configured it to not have any menus.  All it can do is run
Firefox which goes straight to the Intranet home page.

I have configured some hidden key combinations so I can do some
maintenance if necessary.

  Alt-Ctrl-f will run firefox
  Alt-Ctrl-s will run firefox --safe-mode so I can bypass the kiosk mode
  Alt-Ctrl-t will run xterm

I installed the R-Kiosk add-on for Firefox, so the users can't do
anything stupid.

If they exit from Firefox by pressing Alt-F4, they can't do anything
with the window manager, and Firefox will automatically restart.
