kiwi version 2.55 uses udf for creating an install iso now.
This should solve your problem. In order to be able to mount
the iso the udf kernel module needs to be part of the initrd.
I have added that to the boot image descriptions too. You need
to repeat the kiwi --create step in order to allow the changes
to have any effect

v2.55 will be available today... just before the release party
starts :-))
