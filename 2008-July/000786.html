Hello Community

I am happy to announce version 0.4.1 of KIWI-LTSP : LTSP5
implementation on openSUSE using KIWI imaging technology.

Changes from the last release:

- numlock patch from lejo, use "LDM_NUMLOCK=true" in
/srv/tftpboot/KIWI/lts.conf to enable number lock on boot.
- Added backup switch: Run "kiwi-ltsp-setup -b" to backup current
configuration, the back is stored in
/usr/share/kiwi/images/ltsp/suse-11.0/
- Added help switch -h/?, kiwi-ltsp-setup when run with these switches
will display help

For instructions on how to get it working see this: http://en.opensuse.org/LTSP

Apart from kiwi-ltsp release, Lars has created packages for Easy-LTSP,
a GUI for ltsp management. Currently it does not do much, but you will
get the idea of where it is headed. Packages also available for Fedora
9 apart from all current SUSE releases.

The packages available here:
http://download.opensuse.org/repositories/server:/ltsp/

Have a lot of fun...

-J
