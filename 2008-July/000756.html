> Can I add the Update-Repository from openSUSE 10.3 to my config.xml?
> If yes, how I can add the newest update to the live system.

You have multiple possibilities here:

1) you can upgrade your previosly created image root tree (kiwi prepare)
   with kiwi again. Let's say you have prepared your tree below /tmp/kiwi-test
   the following will upgrade the tree using the update repo for 10.3

      kiwi --upgrade /tmp/kiwi-test \
           --add-repo http://download.opensuse.org/update/10.3 \
           --add-repotype rpm-md

   now you need to re-create the image from the updated tree as you
   did before

      kiwi --create /tmp/kiwi-test ...


2) you can use zypper or smart inside the image after the first
   deployment. So if your system runs and you have installed for example
   smart you can add the update repository with 

      smart channel add ...
      smart upgrade 

   depending on the image type the changes might be in RAM only
   ( for example on live ISO's) so this might not be a good idea
