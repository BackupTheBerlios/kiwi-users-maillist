> in create_xconf the flag "RECONFIGURE" is used to determine if the X 
> configuration should be reset.
> Where is $RECONFIGURE set and how can I manipulate this flag at boot time?
> It should be through cmdline, but I cannot find the script mapping the 
> kernel parm to RECONFIGURE, so I would have to do this manually.

Ah well this RECONFIGURE thing is work in progress. I see that it's
important to be able to re-configure but it's much more complex
than it seems to be. At the moment the RECONFIGURE flag was only
implemented in the runlevel script but there is no infrastructure
which set the variable at the moment except if you are booting
via pxe you can plase that RECONFIGURE=yes into your config.<MAC>

It would be easy to add a mapping from the kernel commandline but
it hasn't happen so far. As you know I'm willing to accept
patches :-)
