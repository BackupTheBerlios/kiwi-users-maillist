Hi,

I'm starting to write a more user friendly weekly report
about the changes made in kiwi. I will not mention small
fixes which doesn't have a big impact But changes which
might be of interest or fixing more severe bugs

==================================================
Changes Report kiwi v4.30  2010-04-11 - 2010-04-16
--------------------------------------------------
GIT: changed info module to print XML output

  The kiwi --info option prints some information about
  the image description and the repositories. It's for example
  possible to query a solved list of packages. The output
  of that information is now encapsulated into an XML
  format of the following basic structure:

  <imagescan>
       ...more elements and information
  </imagescan>

GIT: added package version and architecture information when
     requesting the solved packages list via --info

  When calling kiwi --info <image-descr> --select packages
  you will additionally see the package version and architecture


GIT: fixed partedGetPartitionID() function to return correct ID

  When using <type ... bootloader="syslinux"/> the syslinux
  bootloader on a separate FAT partition will be used. There
  was a bug detecting the partition ID of that FAT partition
  which is fixed now. For those of you who use syslinux instead
  of grub this fix is mandatory 

GIT: fixed split boot, cleanImage is not allowed to umount
     the required but non busy symlinked read-write part

  kiwi had a bug that it umounts the non busy filesystems
  before calling init. The split system is based on symlinks
  to another target which can be another filesystem. Because
  of this the other filesystem is never busy but absolutely
  required on order to have a valid root tree. The fix here
  is to prevent the umount in case of a split system image

GIT: speedup prepare step by not removing the repository
     when the chroot switch happens

  This is a performance fix which causes kiwi to download
  repository meta data only once and also leave them on the
  build machine for re-use
