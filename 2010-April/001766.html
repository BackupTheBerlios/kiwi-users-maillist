> I'm looking for a recipe for a pxeboot image for thin clients. It should
> feature a light weight DE, german language support, but most of all support
> for USB scanners. Searching the web was not successful ...
>
> Any hints?

Check out KIWI-LTSP http://en.opensuse.org/LTSP,
http://en.opensuse.org/LTSP/Quick_start

Add SCANNER = True variable in /srv/tftpboot/KIWI/lts.conf.

To customize the prebuilt image see
/usr/share/doc/packages/kiwi/kiwi.pdf from kiwi-doc package.
