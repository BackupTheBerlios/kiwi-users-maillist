On Fri, Apr 23, Marcus Schäfer wrote:

> With the next release of kiwi (4.32) I plan to remove the
> kiwi pattern module (KIWIPattern.pm). This module was used to solve
> the contents of the pattern files provided with the SuSE distribution.
> The module will only be used if all of the following items applies:
> 
>  * the config.xml requested a pattern eg: <opensusePattern name="base"/>
>  * the smart packagemanager is used
>  * there is no libsatsolver available on the image building machine
>    which applies to SuSE <= 10.1 and SLE10
> 
> Are there users who have that environment and therefore still
> rely on the functionality of the KIWIPattern.pm module ? If so
> please drop me a note otherwise I will remove it during the
> next week

The only working kiwi image description I am using (with openSUSE 11.1) is
using that.  So i'd would be very glad, if you do not remove that.
 
I use this:

                <opensusePattern name="kde3"/>
                <opensusePattern name="default"/>

If that's no longer available, I cannot create any images... :-(
