Hi,

> Just wondering...
> 
> I know that kiwi works fine with openSUSE and sles (will certainly be using it later on),
> But is there anybody on the list who is using kiwi for building images outside suse?

kiwi supports RHEL, Centos and Fedora (types: iso,vmx and oem). I received
feedback from users who were able to build that type of images. I myself
also built working RHEL images. The kiwi-templates package also provides
JeOS templates for rhel-05.4 and rhel-06.0

I recently added support for apt/dpkg and debootstrap as a start for
Debian based distributions. But the support is not finished and there
seems to be no interest in the community so far

Unfortunately my time does not allow to test all the others and I'm focused
on SLES/openSUSE for good reasons ;-) so I can only speak for the quality
of the suse images

In principal the project is open for all distributions and the
design of the code reflects this. Help from other distributions is
always very much appreciated
