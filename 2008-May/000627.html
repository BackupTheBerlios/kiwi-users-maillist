> Hi everyone!
> So i'm trying build LTSP  system  with  HP T5135  as thin client.
> All  fine but i have two problems.
> First is that TC video driver, i'm only available boot this TC with VESA,
> I'm tried add to packages openchrome driver(in config.xml), but it does
> appear in the final image

Are openchrome drivers available on the media or
/usr/share/kiwi/images/ltsp/suse-<version>/extra-rpms ?

> Second is keyboard layouts, i'm specified in lts.kiwi layouts what  i need,
> but  this values are totally  ignored.
> Can anybody help me with this.

Try adding the video server and keyboard layout required in
/srv/tftpboot/KIWI/lts.conf

For example:

XKBLAYOUT=de
XSERVER = vesa

See http://doc.ubuntu.com/edubuntu/edubuntu/handbook/C/customizing-thin-client.html
for details.
