> So, I have copied the contents of the ext3 file system to a directory
> and made my changes. Now I need to make an ext3 file system from this
> directory. Just like kiwi did. Any pointers?

I have tried this:

	# origExt3: mounted fsdata.ext3 from kiwi clicfs image

	# rootDirectory: exact copy of file system in fsdata.ext3

	cd origExt3

	cp -a . rootDirectory

	# Make space for 125MB image. The original fsdata.ext3 is 116 MB

	dd if=/dev/zero of=fsdata.ext3 count=250000

	mkfs -t ext3 -q fsdata.ext3

	mount -o loop=/dev/loop3 fsdata.ext3 tDiskMount

	cd rootDirectory

	cp -a . tDiskMount/

	umount tDiskMount

	mkclicfs fsdata.ext3 RSofT-Mini-openSUSE-11.2.i686-1.3.99

I was uncertain if the dd command should be for the size of the
fsdata.ext3 as is, or the size it is after resize2fs. In this session I
did not do a resize2fs command. There were no complaints at all when I
did this. The only question was by mkfs verifying that I was making the
file system on a non-block device.

The image is loaded over PXE. But there are lots of complaints like:

	startprocess: command not found.

So I guess my image is not exactly as expected...
