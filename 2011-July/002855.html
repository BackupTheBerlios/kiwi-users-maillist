Yes, with an OEM image you will not get raid support.

We do have the preHWdetect hook script for an OEM image and you could 
potentially set up your raid array to appear as one disk prior to Kiwi 
looking at the box for installation. For this you'd have to get all the 
raid tools you need into the initrd, this is done by adding the 
bootinclude="true" attribute of the <package> element. This might be an 
option, but I may be missing something.
