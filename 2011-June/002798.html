> I upgrades kiwi and the error stays, so I am doing something wrong.

ok, so you are on the kiwi version from home:ctso, right ?

> I have a working SLEPOS 11 setup working and booting via PXE, using the
> infrastructure provides by SLEPOS (adminserver/branchserver). I now have to
> support usb boot and install, ie, replace PXE with USB and then install the
> same image to the harddisk.
> 
> I modified the image type to usb and rebuilt it. That went fine. I then
> tried to execute kiwi --installstick and got the errors in my last mail.
> 
> I think its because my image is not a 'vmx' image, but I dont know how to
> build one.

The following should work for you:

* edit the config.xml and add:

  <type image="oem" filesystem="ext3" boot="oemboot/suse-SLES11" installstick="true">

* after that call kiwi as follows:

  mkdir /tmp/myimage
  kiwi --build <path-to-config.xml> -d /tmp/myimage --type oem

This will run kiwi prepare and create in one step (build) and
because you have added installstick="true" it will also build the
installstick image file named: /tmp/mytest/....raw.install.raw

Hope that works for you
