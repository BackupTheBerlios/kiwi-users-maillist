I think I have found a problem with the patternType checking and delete type packages sections.
If you have a delete type packages list for a profile (profileA) that is not the one currently being built, and you are using patternType="plusRecommended" for the profile, you get the following error when trying to build other profiles:
Jun-14 15:01:25 <3> : Conflicting patternType attribute values for "profileA" profile

If you try to add the patternType attribute to the delete packages section, you get the following error:
Jun-14 15:01:50 <3> : The patternType attribute is not allowed on a <packages> specification of type delete.

If I remove/comment out the delete section for profileA I can build the other profiles.

Easily reproduced with any of the sample images with multiple profiles by adding a delete section for a profile, then setting patternType="plusRecommended" for all the packages sections for that profile.  Then try to build a different profile

I'm using kiwi 4.88, and 4.91.2 and can reproduce this problem.
