> Hi people.
> I'm trying to deploy memtest86 in kiwi pxe menu. Unfortunately until now I had no sucess with this.
> 
> Did someone already done this? Could you send how?

* add the following in your pxelinux.cfg/default (or host specific) file

  DEFAULT memtest

  ...

  label memtest
     kernel memtest86

* make sure you have the memtest86 binary copied to /srv/tftpboot/
  memtest is provided by the 'memtest86+' package
 
That should do the trick. More information can be found here
http://pxe.dev.aboveaverageurl.com/index.php/PXE_Booting#memtest86
