Hello KIWI users,
I am using the latest KIWI from Virtualization:Appliances repository on 
openSUSE 11.3 system. I am suffering hard from KIWI hanging for more than hour 
during the image build. In the log file I see that it hangs in the umount 
command. After some investigation, I found that this is maybe a problem of the 
kernel. See:
	https://bugzilla.kernel.org/show_bug.cgi?id=15906
if you are interested in details. From the above discussion in the kernel bug 
tracker, I found that adding a simple "sync" call before the "umount" works 
around this.
	I have prepared a patch for the /usr/sbin/kiwi perl script and I am using 
it since a few KIWI releases. It works flawlessly. Although the problem is 
almost surely not in KIWI itself, please include this in the next KIWI release 
if possible. I believe it won't hurt at all?
Best regards,
Todor
