Hi,

> Is there a built in way to remove source control files when building a system image?
> 
> (e.g. any .svn directories and files that exist in the <source>/root directory structure)

There is no built in function for this but the following in
config.sh should do it

  find -type d | grep .svn | xargs rm -rf
