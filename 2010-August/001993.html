Hi,

> Is there a way to run a script while chroot'd inside the final image?  I
> would like to set some extended attributes on the filesystem and this
> seems to only be possible after the output filesystem is created.

There is the preinit step in kiwi. It's inside the new system but
before init was called. For example:

  /usr/share/kiwi/image/oemboot/suse-SLES11/root/preinit

Is this what you are searching for ? Of course if you change
something here you should consider to create a clone of the
oemboot/suse-SLES11 boot image and maintain a suse-pzb-SLES11
version of it :-)
