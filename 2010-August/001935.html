Changes Report kiwi v4.52 2010-07-25 - 2010-07-30
--------------------------------------------------
most important changes for last weeks release...

GIT:  added support for image caching. With this feature you can
      provide a cache directory along with the option --cache.
      kiwi in prepare step looks into the directory if it finds
      a cache serving as the base-root for the system image.
      If it can't find one it creates caches based on the
      patterns used in the image description which speeds
      up subsequent prepare calls for this and other images

  if you call kiwi --prepare ... --cache <some-dir> it will create
  a cache for this build. The cache is splitted up if there are
  patterns in use. In that mode you can re-use the cache for
  other builds too and things should be faster. The initial prepare
  which creates the cache takes longer though. I'm still working
  to improve caching so this feature is not complete yet

GIT:  fixed suse-11.3/suse-min-gnome example, added missing
      icewm and a terminal :)

  @Robert: for 11.3 it didn't install a windowmanager

GIT:  fixed setupBootPartition() to return with a bootid
      export in _any_ case. Default bootid is 1 (bnc #625379)

  most important fix over the 4.50 release. The bootloader
  setup was broken so a reboot didn't work

Have fun :-)
