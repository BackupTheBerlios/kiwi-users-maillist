Hi,

> We use pre-built initrd, I'll take a guess that is the problem.

yes

> Is there a way we can pre-build and still get the setting?

sure you can rebuild the image with checkprebuilt="false" and let
kiwi create the initrd for you. After that you can use that
initrd as a prebuilt again.

Basically the problem with prebuilt exists if you tend to
change parameters which has an impact on the initrd. Mostly
all oem- parameters are evaluted inside the initrd
