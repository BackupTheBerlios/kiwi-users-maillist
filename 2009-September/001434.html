> Thanks for the pointers. Still leaves me with the question, once I put
> my Python script into /usr/share/firstboot/scripts how do I actually get
> it to be executed. The mere presence of the file in the directory is not
> sufficient.

Firstboot mechanism really just takes all scripts from this directory (well, 
exactly from the directory specified as SCRIPT_DIR in 
your /etc/sysconfig/firstboot, but this is the default value) and executes 
them. So if your script is executable directly from shell (executable rights, 
header pointing to interpreter), it should work.
