Hmm, you got a lot of reading to do :)

check out /usr/share/doc/packages/kiwi.pdf from kiwi-docs package.

Check the repositories list in
/usr/share/kiwi/images/yourimagename/suse-11.1/config.xml, it should
have update repository as seen in
/usr/share/kiwi/images/ltsp/suse-11.1/config.xml preference section.

        <repository type="rpm-md">
            <source path="http://download.opensuse.org/update/11.1/"/>
        </repository>

Run "kiwi-ltsp-setup -fat" and "kiwi-ltsp-setup -n" to create netboot
and system images.

You need a good internet connection when building the image.
