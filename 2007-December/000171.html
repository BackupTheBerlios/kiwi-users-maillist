Hi,

> checking EXT2 write extend
> Failed to mount root filesystem
> reboot Exception: reboot in 60 seconds

I guess a problem with the aufs mount. Just repeat the process and
add the following kernel option when the grub screen appears

	kiwidebug=1

after that you will get a shell when the reboot Exception happens.
Now you can change to the error console which is on console 3 where
you should see a lot more information about the failure. On console
4 you can see the errors the kernel throws if any

That's not a good test with a USB stick image because with that emulation
