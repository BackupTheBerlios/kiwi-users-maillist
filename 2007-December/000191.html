Hi Richard,

> > Thanks for your mail I read it and I think one kiwi action might not be
> > clear here. When kiwi builds the system image (that's the big 700MB or so
> > thing) it also requires to access a method to boot it. In case of an ISO
> > image this means kiwi needs to build a boot image too (initrd). The boot
> > image descriptions
> > comes with the kiwi-desc-*boot packages which installs data in
> > /usr/share/kiwi/image/*boot. In your case kiwi picks the boot image to
> > boot an ISO live system from /usr/share/kiwi/image/isoboot and
> > _automatically_ builds it together with your system image. So what you
> > see in the log is a build of a system plus a boot image. The errors
> > you found are related to the boot image build.
> 
> Ah, that's clearer now indeed.
