>> How to insert it to initrd ?
>> It is installed in RO(squashfs) fs in livecd. But there is not it in initrd. 
> 
> 
> To insert into the initrd you need to add <file name="fs/squashfs.ko"/>
> into the drivers section and the package name which contains the kernel
> module into the package section of the boot image description according
> to the image type you want to create. I guess you want to build a live
> system (ISO) so this is the file
> 
>    /usr/share/kiwi/image/isoboot/suse-SLED10-SP1/config.xml
> 
> I still don't know exactly what you want to achieve but if I again guess
> you want to build a live system with squashfs compression you also need
> aufs in order to be able to mount the compressed image together with
> an read write part. aufs also doesn't exist on SLE
> 
> * All this would be a lot easier if you use 10.3, don't know if this is
>   an option for you.
> 
> * By the way you are not forced to use a compressed image you can
>   simply create the live system without compression. Of course your
>   image gets bigger than but you don't have to deal with inclusion of
>   squashfs / aufs into the initrd. To do this you simply remove the
>   flag="unified" from your system image config.xml file
> 
> Regards,
> Marcus
