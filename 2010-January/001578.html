> We use clicfs over AoE/NBD for kiwi-ltsp, works fine.

I have tested this as well now with the latest version of kiwi.
The following works:

   cd /srv/tftpboot/image
   vbladed 0 1 eth2 LimeJeOS-openSUSE-11.2.i686-1.11.2

   ==> LimeJeOS-openSUSE-11.2.i686-1.11.2 is a clicfs file
       created by kiwi with the following type:

       <type filesystem="clicfs" boot="netboot/suse-11.2">pxe</type>

   config.<MAC>

   AOEROOT=/dev/etherd/e0.1

   ==> boot the client, copy on write actions happens
       into RAM
