Hi,

> is it possible to create a swap partion when creating a vmx image? How do I
> configure this in the config.xml? 

swapping in a virtual machine might not solve the problem. Anyway plain
vmx type does not support creating a swap partition. Switch to the
oem image type and you will get a swap partition. You should specify
the following oem options:

    <oem-systemsize>...</oem-systemsize>
    <oem-home>false</oem-home>
    <oem-swapsize>...</oem-swapsize>

values are interpreted as MB
